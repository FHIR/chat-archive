Lloyd McKenzie (Feb 26 2020 at 20:19):

Grouping is pretty decoupled from proposed disposition. Grouping is things like "Fit-for-teleconference" or a project name (e.g. SDC, Workflow) as well as things like "Ready-for-vote" or "Block-Vote-1". You can change groupings without proposing a resolution using the "Adjust Fields" workflow. (You can modify them using 'edit' capabilities too, but that only works if you've already got a grouping - and it's soon going to go away because 'edit' creates other issues...)
