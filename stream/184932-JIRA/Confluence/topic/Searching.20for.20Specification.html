Patrick Werner (Nov 18 2019 at 07:12):

Just checked out the new Jira Tracker an tried to search for all tracker items for a Work Group, which works fine. But when i try to get all trackers for a Specification, it doesn't work.
The search term Specification is recognized, but there is no auto-complete on the value, and nothing is found.
Example: Specification = "Genomics Reporting (FHIR)" doesn't return any issues.

Lloyd McKenzie (Nov 18 2019 at 10:59):

There are a few fields in Jira that are 'look-up' fields. To get search on those to work, you need to fill in the pre-requisites. To get workgroup or specification to function, you have to select the project (FHIR) and the issue types (all 4). To filter by artifacts or pages, you need project, issue type and specification selected. (I believe this is covered in the documentation. If you can't find it, holler and I'll update.)

For example, this query gives me 320 hits:
project = FHIR AND issuetype in ("Change Request", Comment, Question, "Technical Correction") AND Specification = "Genomics Reporting (FHIR) [FHIR-genomics-reporting]"
