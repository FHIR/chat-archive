Topic: Rights to comment?

Bryn Rhodes (Apr 23 2020 at 19:21):

Do JIRA users need particular rights to be able to comment on tickets?

Lloyd McKenzie (Apr 23 2020 at 20:41):

Anyone should be able to comment on new tickets so long as they have an account. If it's resolved to "change required", then managers can add comments. If it's resolved to something else or moved to Change Applied or Published, only admins can add comments

Lloyd McKenzie (Apr 23 2020 at 20:42):

Logic being that we don't want newcomers putting comments on resolved stuff that will never be applied. Managers might provide comments on approve stuff about how to apply the change, but that's not usually appropriate from most folks (need to keep in mind the scope of the approved change when providing feedback).

Lloyd McKenzie (Apr 23 2020 at 20:42):

If there are concerns about that set of rules, holler...

Lloyd McKenzie (Apr 23 2020 at 20:42):
