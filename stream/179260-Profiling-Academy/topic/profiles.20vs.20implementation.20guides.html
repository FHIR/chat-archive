Nelo Aguila (Nov 18 2020 at 01:06):

Hi. Complete newbie here. Can someone point out the difference between Profiles and Implementation guides? I was thinking they were one and the same but I found out two separate registries for each one and got confused. Thank you!

Lilian Minne (Nov 19 2020 at 07:34):

Hi @Nelo Aguila a profile is a set of constraints on a FHIR base model (or data type or another profile), while an Implementation Guide is a guide for implementers that describes how they should implement FHIR in a specific use case. So an Implementation Guide may include multiple FHIR profiles, extensions and examples around a specific use case.
