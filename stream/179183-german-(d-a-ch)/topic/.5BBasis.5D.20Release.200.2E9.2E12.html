Topic: [Basis] Release 0.9.12

Alexander Zautke (Dec 17 2020 at 19:43):

Das TC FHIR hat soeben die Version 0.9.12 der Deutschen Basisprofile veröffentlicht. Dieses Release enthält eine Reihe von technischen Korrekturen wodurch die Validierung mit dem offiziellen Java Validator nun reibungsloser funktionieren sollte. Der ausführliche Change-Log findet sich hier: https://simplifier.net/packages/de.basisprofil.r4/0.9.12

Vielen Dank an alle die in den letzten Monaten an diesem Release mitgearbeitet haben!

Lars Heyden (Dec 18 2020 at 06:55):

Wird dieses Release als stable gekennzeichnet? Oder ist die letzte stabile Version weiterhin 0.9.2?

Alexander Zautke (Dec 21 2020 at 14:31):

Vor der Version 1.0.0 (Geplantes Release März 2021) wird es kein offizielles Stable Release mehr geben. Die Version 0.9.12 sollte jedoch in allen Fällen anstatt der Version 0.9.2 verwendet werden.
