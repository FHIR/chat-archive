Topic: [Release] Deutsche Basisprofile 0.9.13

Alexander Zautke (Jan 29 2021 at 15:46):

Das TC FHIR hat soeben eine neue Version der Deutschen Basisprofile freigegeben. Das Update steht unter https://simplifier.net/packages/de.basisprofil.r4/0.9.13 zur Verfügung. Dieses Release enthält eine wichtige Änderung: Alle Identifier-Profile haben nun unter Identifier.type ein Slice By Pattern. Ansonsten bietet dieses Release nun Support für die KZVAbrechnungsnummer, definiert durch die Kassenzahnärztliche Bundesvereinigung.
