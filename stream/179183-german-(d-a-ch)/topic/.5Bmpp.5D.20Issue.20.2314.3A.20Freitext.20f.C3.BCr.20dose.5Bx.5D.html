Simone Heckmann (Sep 25 2017 at 14:35):

MedicationStatement.dosage[x] kann entweder vom Typ doseRange oder doseQuantity sein. Letzteres wiederum kann entweder strukturiert mit value, system, code als Wert mit codierter Einheit oder einfach mit "unit" als String angegeben werden: http://build.fhir.org/datatypes.html#quantity

Simone Heckmann (Sep 25 2017 at 14:38):

dosageRange verwendet den gleichen Datentyp, dann jedoch gibt es eine Quantity für "low" und eine für "high", beides wiederum kann über Quantity.unit als Freitext ausgedrückt werden.
