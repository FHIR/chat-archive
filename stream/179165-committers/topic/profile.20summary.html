The build generates a summary tab for describing the profile. The content of this is built automatically. Profile editors can now include their own text by either
- (if you're using spreadsheets) making a metadata entry "summary-"[sheetname] where tab name is the name the spreadsheet
- (if you're authoring a profile) adding an extension http://hl7.org/fhir/StructureDefinition/structuredefinition-summary which is of type 'markdown' to the profile (StructureDefinition)
