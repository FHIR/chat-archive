  • Some resources make no mention of workflow (even though they are listed as workflow resources)
  • Some resources have their first sentence in the "Scope and Usage" section stating something like "This resource is an event resource from a FHIR workflow perspective - see Workflow."

Thoughts on whether we should all try to be consistent with either including/excluding text? If included, preferences on where the blurb should reside and what it should say?
