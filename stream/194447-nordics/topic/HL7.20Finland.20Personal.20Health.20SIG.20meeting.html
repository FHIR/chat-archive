Mikael Rinnetmäki (Oct 07 2019 at 16:18):

Hi all,
HL7 Finland arranges a meeting on Tuesday November 26 2019 in Helsinki, with the following (preliminary) agenda:

Special guest lecturer: Grahame Grieve
9:00 Personal Health SIG
Design competition for the MyKanta PHR – results, app presentations and experiences from working MyKanta PHR (Ministry of Social Affairs and Health, competition winners)
Latest from the MyKanta PHR (Kela / THL) 10:15 Coffee

10:30 Overview of the current state of FHIR - What's new and what's coming (GG) 12:00 Lunch
12:45 FHIR education continues (GG)
14:30 Coffee break
15:00 FHIR education continues (GG)
16:30 Closing remarks

The educational session will touch these areas of FHIR:
• FHIR in personal health - patient access / patient merge / personal health record
• CDA/v2/FHIR / mapping / translation – migration and parallel use
• implementation guides / profiles / validation / FHIR community process / building FHIR
communities
• graphql / efficient API use / light data analytics on the API directly
• multiple FHIR versions, languages and timezones / API safety
• FHIR and social services (/social determinants for health)
• FHIR and SNOMED CT
• FHIR and secondary use of patient data
• FHIR connectathons, demos and interoperability showcases
• National initiatives in using FHIR
