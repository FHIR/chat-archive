A couple of weeks ago, on the IG publisher/templates call, there was a discussion about migrating IGs between tools. Overall, the feeling was that, aside from the actual conformance resource content, there was no expectation to be able to move an IG from one toolset to another. However, @Grahame Grieve pointed out that there is a requirement to be able to run the output of whatever tool through the IG publisher if you want the IG published by HL7 in order to perform various validation checks.

If I've captured the discussion correctly, is it currently possible to export a "rendered" IG from Simplifier and feed it into IG publisher? I can see ways to download various a project zips, but none correspond to a "build" IG suitable for running though the publisher for validation checks, nor do they correspond to something that can be fed into IG publisher to build an IG. Did I miss a download option? (I am logged into Simplifier, and an admin on the project I'm looking at.)
