for Observation>status - added a comment to also process status of amended and corrected.

Linda (Aug 15 2018 at 15:33):

for Practitioner, Anne felt we need a list of qualified providers to do this so we are using DEQM Practitioner>Qualification>code and there will be a value set for the types that meet the requirement. To do this, we had to change the binding to the Qualification>code to use the code set specified as an example but change binding to required, i.e. Binding: v2 table 0360, Version 2.7 (required)

Linda (Aug 15 2018 at 15:35):

Note that we did NOT change qualification to a required element. Receivers of the data can decide if they need that field for processing the measure or have other ways to obtain that concept within their system.
