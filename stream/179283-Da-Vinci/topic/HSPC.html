David Hay (Sep 30 2019 at 03:01):

follow these steps (if you haven't followed them already :)

  1. load clinfhir.com
  2. at the top right (under current servers) click 'Add server', then paste 'https://api.logicahealth.org/DaVinciCDexPayer/open/ into the url box (be sure to keep the trailing /)
  3. click the 'Test' button that appears to the lower right in the dialog. All going well you should get a message saying that the server can be added. Add a name for the server, select the FHIR version as R4 and save. You'll return to the main screen.
  4. Now, click the 'edit' link in the top right. You should be able to select the server you just added as the data server. Do so, and save.

Now you can load the Patient View app, and you will be accessing the server that Viet has mentioned, and you just added. I was able to add a new patient and view their resources.

Let me know if you still have issues.
