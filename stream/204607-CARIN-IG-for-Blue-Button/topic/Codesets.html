Pat Taylor (Jun 21 2021 at 14:26):

@Suma Addagadde The CARIN IG uses definitions defined by the HL7 Terminology Authority. The TA provides definitions based on information from the code system's owner, who in this case, is he CDC. Many payers, including CMS, do not include the decimal point. HL7 has defined the Code System: https://terminology.hl7.org/2.1.0/CodeSystem-icd10CM.html. Although it is copyrighted, use of the codes does not require a license. @Amol Vyas @shaifali
