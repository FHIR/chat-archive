Isuru Samaranayake (Feb 12 2021 at 17:25):

Hi all, I'm new to CARIN IG and having a confusion regarding how to request a CARIN EoB from a server. When we are following CARIN IG, the "_profile" search parameter is mandatory or can we request EoB without "_profile" parameter?

Corey Spears (Feb 12 2021 at 19:10):

The CARIN for Blue Button IG does not specify the use of a _profile search parameter. It is not a requirement for a CARIN BB IG server to support _profile, nor is it a requirement for a conformant client to request an EoB with it.
