Melissa Benzie (Aug 06 2021 at 13:38):

The Carin Stu1 IG defines the patient discharge status code system Uri differently than what is listed on the Fhir Terminology Code Systems https://www.hl7.org/fhir/terminologies-systems.html

The supportingInfo:discharge-status slice code element has a required binding to the ValueSet: NUBC Patient Discharge Status Codes. The system uri for the valueset is defined as: https://www.nubc.org/CodeSystem/PatDischargeStatus. This is different than the uri specified in the Fhir terminology systems https://www.hl7.org/fhir/terminologies-systems.html, which defines it as: http://www.nubc.org/patient-discharge

Corey Spears (Aug 06 2021 at 19:27):

I will raise the issue with the HTA.
