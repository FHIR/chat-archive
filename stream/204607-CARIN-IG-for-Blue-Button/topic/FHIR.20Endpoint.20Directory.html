Ryan Howells (May 06 2021 at 16:36):

Thanks for your comments @Richard Braman (FLY.HEALTH) @Josh Mandel. At CARIN, we anticipate multiple private sector end point directories in the market. The ONC and MITRE end point directory provides an open framework for others to use. It's our understanding thought that the ONC/MITRE directory will not be proactively capturing payer end points. Unfortunately, the CAQH press release wasn't clear. It seemed to indicate that CARIN endorsed the CAQH end point directory as the only one payers should use. It should be noted that we did not approve the language in the press release before it was issued. It also should be noted that since we began, CARIN does not endorse any specific vendor's product or service offering. We are vendor, technology, and product agnostic. We always have been and always will be. Our hope is the industry can self-organize around a common approach for developing an end point directory the everyone can follow. We would also agree being listed in the end point directory and having access to that information should be a public good otherwise it creates perverse incentives in the market. @Richard Braman (FLY.HEALTH) we would welcome the opportunity to chat about the directory you have put together. Can you ping me offline so we can set-up a time to chat?
