John Achoukian (Oct 16 2020 at 17:49):

Practitioner.qualification addresses official certifications, training, and licenses. The example provided focuses exclusively on Degrees. How would this section distinguish another qualification code set (e.g. FSMB, state boards, credentialing organizations, etc.)? @Amol Vyas

Josh Lamb (Oct 16 2020 at 18:25):

practitioner.Qualification is not flagged with a "must support". The use of this field would be outside of this IG.

You can take a look at the PlanNet IG (or another directory implementation guide) for examples of how other guides are populating this field. Here is link to Davinci PlanNet IG: https://build.fhir.org/ig/HL7/davinci-pdex-plan-net/.
