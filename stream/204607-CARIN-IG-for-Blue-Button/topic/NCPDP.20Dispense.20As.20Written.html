The system uri for the NCPDP Dispense As Written that is specified in the Carin IG is different than the HL7 Terminology published code system uri.

Carin IG code system uri: http://terminology.hl7.org/CodeSystem/NCPDPDispensedAsWrittenOrProductSelectionCode
defined here: (https://build.fhir.org/ig/HL7/carin-bb/branches/v1.1.0/CodeSystem-NCPDPDispensedAsWrittenOrProductSelectionCode.html)

HL7 Terminology system uri: http://terminology.hl7.org/CodeSystem/NCPDP/DispensedAsWrittenOrProductSelectionCode
defined here: (https://terminology.hl7.org/2.1.0/CodeSystem-NCPDPDispensedAsWrittenOrProductSelectionCode.html)

Corey Spears (Oct 19 2021 at 22:07):

This was changed in response to https://jira.hl7.org/browse/FHIR-31492 upon the request of the HTA.
It has been updated on the Confluence page: https://confluence.hl7.org/display/TA/NCPDP+Dispensed+As+Written+Or+Product+Selection+Code
It is also in the CI Build for terminology.hl7 (THO): http://build.fhir.org/ig/HL7/UTG/CodeSystem-NCPDPDispensedAsWrittenOrProductSelectionCode.html
I believe it should be in the next THO publication.
