Topic: Creating a Jira Issue (tracker) against the CarinBB IG

Saul Kravitz (Dec 09 2020 at 20:23):

It is a very safe bet that the current IG is not perfect. If you identify shortcomings, the surest way to ensure that your input receives serious consideration when the IG is updated is to create an issue (sometimes called a tracker) in the HL7 jira -- https://jira.hl7.org/. This requires login credentials. You can see all of the CarinBB IG issues through this view -- https://jira.hl7.org/issues/?filter=12825 .

If you want to create a new issue, click on the light blue Create button in the menu bar.
Then choose the CarinBB Specification (if you type 'carin' or 'US' it will be suggested), and pick version 1.0.0.
