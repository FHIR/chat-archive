Topic: RxEOB Product Type - HRI and UPC

Kate Dech (Jun 25 2021 at 20:04):

As a claim payor for Pharmacy, we support multiple product types in addition to NDC. We also support UPC and HRI for a limited set of products that are payable under the Pharmacy Benefit such Diabetic Devices and Supplies and some OTC Products. Currently the Pharmacy EOB item.productOrService must be either the compound code or be an NDC in the FHIR NDC code system.

These are also 11 digit values on the payer side so a product type code is used to ensure unique codes.

Examples:
UPC 20221-0260-06 (EXCEL COMFORT POINT INSULIN PEN NEEDLES 31G X 4MM)
refer to https://www.gs1.org/gtin

HRI 08317-7611-03 (GLUCOCARD VITAL BLOOD GLUCOSE MONITORING SYSTEM)
refer to https://healthdata.gov/dataset/NHRIC-National-Health-Related-Items-Code-/atu2-gaky
