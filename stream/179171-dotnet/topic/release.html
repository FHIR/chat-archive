z Brian Postlethwaite (old - don't use) (Jan 07 2016 at 12:25):

Hi all dot net devs out there,
There is a new release *0.90.4* of the dotnet FHIR client now available on NuGet.
There is a version for both DSTU2, and DSTU2.1(Jan Connectathon build) - And compatible Specification Project too.

This has been a big release with the generation of the model classes being totally re-written using t4 templates based on the resource profile XML files from the build.

_Release Notes:_
http://ewoutkramer.github.io/fhir-net-api/whats-new.html

_Nuget Packages:_
http://www.nuget.org/packages/Hl7.Fhir.DSTU2/
http://www.nuget.org/packages/Hl7.Fhir.DSTU21/

_Note:_
The full source is as always on github, in either the *develop* or *ft-connectathon-jan2016*

Thanks to all those who worked on the release!
