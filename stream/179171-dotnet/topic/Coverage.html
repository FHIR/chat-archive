Angus Millar (Aug 21 2019 at 22:41):

Perhaps you mean the identifier property of the reference data type, like this:

Coverage Cov = new Coverage();
Cov.Payor = new List<ResourceReference>();
var Ref = new ResourceReference();
Ref.Identifier = new Identifier ("http://some.code.system.uri", "123456");
Cov.Payor.Add(Ref);
