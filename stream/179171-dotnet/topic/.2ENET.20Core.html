Brian Postlethwaite (Jul 04 2016 at 07:31):

If you want to build locally and update the models at any point,
1. Get the latest source from the above branch from GitHub
2. Run the Powershell script DownloadFhirBuildProfile.ps1 from the build folder
(This will download all the required profile files and example zips - ~80MB, so takes a while)
3. Open the src\HL7.Fhir.sln
4. Run the t4 templates in the Model\Generated folder inside Visual Studio
(in the HL7.Fhir.Core.Net45, and the other 2 projects also)
5. You can then check the diffs in git to see that only new changes have come in.
6. Build the solution!
