We've improved the fhir-net-api 1.5.0-beta1 and we now have a final version of 1.5.0!

This release contains the technical corrections for FHIR 3.0.2 and 4.0.1, together with some general bugfixes/improvements.

For a complete list, please visit the pages with the release notes:

  • STU3: https://github.com/FirelyTeam/fhir-net-api/releases/tag/v1.5.0-stu3
  • R4: https://github.com/FirelyTeam/fhir-net-api/releases/tag/v1.5.0-r4

There is also a 1.5.0 for DSTU2 but that is there to retain binary compatibility with the newer shared assemblies but has none of the new features.
