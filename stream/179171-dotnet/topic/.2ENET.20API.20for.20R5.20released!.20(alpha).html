Topic: .NET API for R5 released! (alpha)

Marco Visser (Jan 30 2020 at 16:05):

We have just released the first alpha version of the FHIR R5 .NET API, allowing you to build a FHIR R5 client in .NET. This alpha release implements R5-preview #1.

You can find the packages for this alpha release on MyGet: https://www.myget.org/F/fhir-net-api/api/v3/index.json (not on NuGet!!)

As icing on the cake, we have also released a new Vonk release with R5. Come and see it here: http://labs.vonk.fire.ly. This server can be used on the connectathon in Sydney.

Thanks @Brian Postlethwaite for helping us to create this R5 alpha release.
