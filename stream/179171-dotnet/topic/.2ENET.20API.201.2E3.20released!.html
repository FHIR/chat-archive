Topic: .NET API 1.3 released!

Ewout Kramer (Jul 11 2019 at 09:41):

First off, congratulations to us all as a community: the .NET API NuGet packages have surpassed the magic 1M downloads mark, thanks to your continued support and enthousiasm. The Firely team really enjoys supporting this project that is obviously useful to a lot of us!

As a bonus, we'd like to announce that we have just released the new 1.3.0 version of the FHIR .NET API. This is mostly a bugfix release, but we have also included noteable new functionality like support for validating custom resources and performance improvements for the DirectorySource. You can read the full releases notes here.

Of course, you can find the binary distribution on NuGet and the sources for the master branches on GitHub.
