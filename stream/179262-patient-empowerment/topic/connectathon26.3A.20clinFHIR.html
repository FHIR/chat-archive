David Hay (Jan 14 2021 at 01:06):

Someone asked on the call about the location of clinFHIR - take a look at http://gb2.clinfhir.com/#id1609884201264 for an example of a graph I created for scenario 1.

To download it into your browser, click the 'Show download screen' and either 'clone' or 'copy' (clone creates new resource id's so probably the best for now).

It's the first scenario and has 3 views representing the stages of the scenario- should be self explanatory (I hope :) )

Perhaps the best overall view is the 'Show project summary' link in the left pane - this shows all the resources in the project broken down by view. If you click on the resource you will see the FSH for that resource in each view

Once downloaded, you can make any changes you like. If you save back to the library it will be a new library entry (You will update the original project if you selected 'copy' during download.

I also saved it to the hapi R4 server - patient id is cfsb16098876468130 . I saved each view in turn so that the Task versions would update appropriately.

If you load clinFHIR with the data server set to hapi4, then the Patient viewer module, then select by id you'll see the resources.

If you check out the task, you'll see the versions there as well...
