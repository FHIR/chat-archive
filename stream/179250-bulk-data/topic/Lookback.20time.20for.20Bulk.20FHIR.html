Topic: Lookback time for Bulk FHIR

ssubash (May 05 2021 at 00:19):

is there a certain lookback time that is specified as part of the FHIR bulk API specifications? For example are there constraints/rules on how far back in the timeline should I be able to apply the Bulk FHIR api.

Lloyd McKenzie (May 05 2021 at 03:15):

In principle, it goes for the length of time you've got records for. Some systems might choose to impose a limit, but there isn't one intrinsic in the spec. In theory, if a system had digitized their patient records back from the 1800s, those could be returned. (If anyone actually has this - let the community know, because that would be amazing :smile: )
