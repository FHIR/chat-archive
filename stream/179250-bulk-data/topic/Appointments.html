Hi All,
Do you know if it's possible to bulk query all the appointments that were booked in a practice?
(It's not displayed on https://bulk-data.smarthealthit.org/)

Josh Mandel (Dec 19 2019 at 23:47):

Re-posting my response here after private e-mail exchange: Currently the FHIR Bulk Data API is defined to provide access to any FHIR resources -- so as far as the API definition is concerned, this is workable. But no real-world EHR has any production support for the Bulk API today, so it's probably too early for you to think about consuming data through this route.

Lior (Dec 20 2019 at 21:06):

Josh Mandel (Dec 21 2019 at 15:29):

Not formally; but I expect many vendors are waiting to see what requirements emerge in ONC's final rule.
