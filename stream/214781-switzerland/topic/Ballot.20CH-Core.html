Oliver Egger (Feb 15 2021 at 07:26):

If you have not yet registered to the CH Core ballot you need to do by tonight by sending me an e-mail.

The CH Core STU2 implementation guide has been published.

You can find the changes compared to STU1 here.

At the bottom of every page you will find a 'Propose a change' link and we look forward to specific feedback on how we can improve the implementation guide, regardless of how you vote for the ballot.

If you do not agree with the changes and you answer the ballot with no, you have to justify this with feedback, please refer to the 'Propose a change' issues that you have entered, you can find them as Github Issues here. We will then try to resolve any relevant negative comments, the process is also described under Ballot rules with the schedule here.

In parallel, the informative ballot for CH EMED and CH VACD are running in cooperation with eHealth Suisse, please send your feedback there also via the respective Propose-a-change link by March 15, 2021.
