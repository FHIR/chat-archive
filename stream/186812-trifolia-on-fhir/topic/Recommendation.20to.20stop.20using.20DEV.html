Topic: Recommendation to stop using DEV

Sean McIlvenna (Dec 18 2019 at 18:37):

In the recent months, people have been using the DEV installation (https://trifolia-fhir-dev.lantanagroup.com) of ToF as the primary installation because the production installation (https://trifolia-fhir.lantanagroup.com) has been so outdated. Last week, we updated the production installation with the latest changes from the DEV environment. Please resume using the production environment as your primary environment, as we want to resume using DEV as a dev-only environment. Sometime near the end of next week we will be updating DEV with some potentially breaking changes. Please ensure that you are using the production (stable) environment by then.
