Topic: Adding an extension to a profile

Katie Wheatley (Dec 31 2019 at 14:41):

I would like to add an extension into a profile at Patient.identifier. I have created both the Patient profile and the Extension correctly in Trifolia and they are both valid. How would I do this please?

Sean McIlvenna (Jan 08 2020 at 19:55):

If both the profile for patient and the extension exist, in the profile for the patient, you should be able to select "extension", constrain it, make a slice on it (so that you can indicate that other extensions are allowed) and then specify a "type" on the extension slice that points to the extension.
