If a Q into a language which is different from the source Q, does that translated Q's id, url and version change?

Lloyd McKenzie (Jul 27 2018 at 21:30):

The id would have to change if it's on the same server as the original. Presumably you'd want the URL and version to remain the same so that the responses could be compared. And that's going to be problematic. So it'd be better to update the original questionnaire to add in the additional languages. If that's not possible, you may have no choice but to create a new URL.
