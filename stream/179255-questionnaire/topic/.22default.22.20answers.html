Topic: "default" answers

Paul Lynch (Aug 28 2019 at 15:20):

Is there a concept of a "default" answer which gets sent if the user submits a form with a field blank? We have initial.value[x], which causes a form to start with fields set to certain values, but presumably the user could clear those value away if the question is optional. (I am not requesting this kind of default; just asking).

Lloyd McKenzie (Aug 28 2019 at 17:07):

If it's optional and the user wipes the 'initial', then presumably the user doesn't want the 'initial'. If you wanted to mandate that an answer be present, then the question should be 'required'.
