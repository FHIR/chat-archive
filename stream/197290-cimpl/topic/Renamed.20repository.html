Mark Kramer (Aug 23 2019 at 12:13):

The cimpl specification repository has been renamed from shr_spec to shr-spec (underscore to dash). This makes the repo name consistent with other standardhealth repos and our filename conventions. This change should be relatively transparent because Github automatically redirects https://github.com/standardhealth/shr_spec to https://github.com/standardhealth/shr-spec. We're in the process of updating the documentation to reflect this change.
