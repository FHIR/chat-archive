Chris Moesel (Oct 07 2019 at 15:51):

CIMPL 6.9.0 is released with the following features:

  • Authoring: ability to constrain a property on an entry that is a choice to a subset of choices.
  • FHIR Export: support dependencies on other IGs specified by user.
  • ModelDoc: Fix cardinality on choice options to 0..1
  • ModelDoc: Display constraints inherited from ancestors
  • Data Dictionary: fix bug which suppressed display of quantity units
  • Data Dictionary: fix value set name in data elements tab

See release notes for details.

Note that it also has changed GitHub repositories to a new shr-tools monorepo.

https://github.com/standardhealth/shr-tools/releases/tag/shr-cli%406.9.0
