@Kurt Allen Not left out, just named differently. It is called DiagnosticReport.Observation. You can determine this because of the mapping statement in the DiagnosticReport class (R4), Observation maps to result (slice on = target.reference.resolve(); slice on type = profile; slice strategy = includes). There is no need to have an element called result that does nothing more than point to an Observation.
