Eric Haas (Aug 12 2019 at 05:38):

Brett and I prepared US Core Block Vote #1 for this Thursday's August 15th 11-12 AM ET HL7 Structured Documents call. These 30 items are tracker items on US Core and represent many issues discussed on the Argonaut R4 Project Calls as well as other trackers brought up by members of the HL7 community. Where applicable we have provided links to the proposed changes so you can view the actual changes in the context of the guide.

Note that this block vote is part of an STU update of US Core in preparation for the proposed ONC U.S. Core Data for Interoperability (USCDI). The Road Map for the STU update is as follows:

  • Review and reconcile 30+ US Core gForge trackers - 8/22
  • Publish fixed STU update version of IG - 9/3
  • 2 week comment period - 9/3-9/174
  • Reconcile STU comments - 9/26 -10/10
  • Publish Final Content- 10/24

As you can see it is compressed timeline and as such the cut off for addressing any new US Core comments (excepting those currently being discussed on the Argonaut calls) is the end of this week. ( Also note that we will be doing editorial QA on the ig prior to publishing to eliminate bad links and typos so rather than creating trackers on those directly email either Brett or myself)

Please respond to this stream if you want to remove an item from the block vote to discuss on the call or if there is a particular comment you want to make regarding a proposed disposition.
