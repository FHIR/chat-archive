Linda (Aug 20 2021 at 01:47):

Da VInci has a group working on Risk Adjustment. Our first focus is on the Risk Adjustment Coding Gap Report going from payer to provider. For that we are using the MeasureReport Resource and the code under Group is where we represent the HCC code. Use case link - https://confluence.hl7.org/display/DVP/Risk+Adjustment and the RA Implementation Guide in development , https://build.fhir.org/ig/HL7/davinci-ra/index.html

I think @Daniel Tam question is slightly different. Since it is a Hierarchical Condition Code, I can see the logic to representing it as a Condition but there is not a code system for them yet. We are discussing that. Daniel you can find the info for our calls on Thursday if that interests you
