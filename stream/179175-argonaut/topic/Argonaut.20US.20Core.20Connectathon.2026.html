Isaac Vetter (Jan 07 2021 at 14:42):

Hi Argonauts! We're not formally participating in this track this cycle, but wanted to share the location of a few Epic FHIR servers, in case they're helpful to others.

  • https://fhir.epic.com/interconnect-fhir-oauth/api/FHIR/R4/metadata
    This FHIR server is generally available all the time, even outside of connectathons. You need to create a developer account and a client id to access it. There's some really great documentation and information on this site.

  • https://apporchard.epic.com/interconnect-aocurprd-oauth/api/FHIR/R4
    Of course, if you're already enrolled in the App Orchard developer program, you've likely already created a client id. This is a great way to test R4.

  • https://connectathon.epic.com/Interconnect-Fhir-Unsecure/api/FHIR/R4/metadata
    Lastly, we're temporarily turning on a temporary and less stable FHIR server, that may be unavailable as we hack away during the connectathon.
