Eric Haas (Jan 13 2022 at 04:54):

Note:

  1. the {{patient_id}} variable is assigned after the patient is fetched
  2. the {{base_url}} variable is assigned from the environment if that is empty or undefined it defaults to theHAPI R4 reference server
  3. the {{validator_url}} variable is assigned to Grahame's R4 reference and used to validated the first resource in the search bundle with the $validate operation.
  4. Currently, we do not have a reference implementation that validate all the latest US Core Profiles
  5. Currently. we do not have a reference implementation that supports all the custom search parameters
