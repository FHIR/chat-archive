Craig Newman (Feb 26 2020 at 13:31):

@Ayaz there is a v2 to FHIR project (see https://confluence.hl7.org/display/OO/2-To-FHIR+Project). So far we have worked on proof of concept mappings of a few of the more common v2 message types (ADT, orders, results, vaccinations). There are a few translation engines being developed. Please have a look at the project and see if that helps.

Caitlin Voegele (Mar 10 2020 at 14:37):

Hi Ayaz - We (Microsoft) just announced an open source project that takes in an HL7 v2 message and leverages handlebars templates to produce a json FHIR bundle output. The tool is seeded with some starting templates that were generated based on the 2-To-FHIR-Project that Craig mentioned above. You can see the project here (https://github.com/microsoft/FHIR-Converter) and let me know if you have any questions!
