Nathan Bunker (Sep 08 2020 at 15:54):

I am planning to verify the VXU to FHIR transformations, with a primary goal of creating a large number of JSON and XML artifacts that are community might use for creating demonstration systems for supporting FHIR. This fall/autumn we have a student project that is going to demonstrate how the Immunization community might move from HL7 v2 to FHIR. This project is going to convert a "reference implementation" that supports HL7 v2 into FHIR. I am going to use resources from this v2-to-fhir project to give examples the students can use. I'll use the time here at the Connectathon to prepare these messages and give feedback. I'm looking forward to working with all of you!
