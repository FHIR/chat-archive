Brian Alper (Jan 27 2021 at 19:20):

The Evidence Resource in R4 is not the structure where PICO ontology will be best mapped. The Evidence Resource in R5 (and soon R4B) uses Evidence.variableDefinition with elements description, note, variableRole (to code the role eg population or exposure), observed, intended, and directnessMatch. We then use EvidenceVariable Resources for the specific variable definitions, but can use Group Resource when the variable role is population or subpopulation to coordinate with other FHIR population descriptions using Group. The EvidenceVariable and Group Resources could be mapped to PICO ontology instances.
