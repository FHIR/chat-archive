Ken Omondi (May 05 2021 at 01:14):

Hi, how do I configure a local HAPI FHIR JPA server to allow auto-creation of reference resources if they don't exist?

Ian Bacher (May 05 2021 at 15:28):

Assuming you're using the JPA Starter, in application.yaml set the hapi.fhir.auto_create_placeholder_reference_targets property to true (it's one of the several commented-out settings).
