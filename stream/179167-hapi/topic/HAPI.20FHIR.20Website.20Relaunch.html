James Agnew (Dec 18 2019 at 11:59):

Hi All,

Over the last 4 months we have been slowly but surely working on a project to rebuild the HAPI FHIR website, replacing the old Maven template with something more modern and usable.

So, good news.. The new site is finally live at
https://hapifhir.io

Among other things, the HAPI FHIR documentation has been completely rewritten, including an all new section on the new interceptor framework and lots of new detail on validation. We're now using a markdown-based format which should make it easier for people to contribute - See the docs sources if you're curious ( https://github.com/jamesagnew/hapi-fhir/tree/master/hapi-fhir-docs/src/main/resources/ca/uhn/hapi/fhir/docs ). The JavaDocs have also been fixed and modernized, as they have been broken for a while.
We've also finally launched our Global Atlas on the homepage. If your project isn't already on it, we'd love for it to be! There is a link on the homepage showing how to submit.

Finally, for the curious, we're open sourcing our style guide too: https://github.com/jamesagnew/hapi-fhir/wiki/Website-Style-Guide

Feedback is welcome!
