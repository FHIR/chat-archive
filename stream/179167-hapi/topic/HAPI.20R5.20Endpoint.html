If anyone is interested in living dangerously... HAPI FHIR now supports the draft R5 definitions on a new endpoint:

http://hapi.fhir.org/baseR5/

Note that this is really only intended for standards developers at this point, as the R5 definitions are likely to change in various ways as R5 evolves. When fields get renamed, any data stored under the old name will magically disappear from the database.
