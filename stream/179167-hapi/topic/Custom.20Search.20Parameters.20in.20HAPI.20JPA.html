Topic: Custom Search Parameters in HAPI JPA

James Agnew (Feb 09 2017 at 00:45):

Hi All- Just a quick note to say that one of the most requested features over the last year has finally been rolled into HAPI's JPA server module: custom resource providers. It's now possible to upload a SearchParameter resource have have its path indexed automatically.

This is deployed now to http://fhirtest.uhn.ca and to the current 2.3 snapshot build. I'm expecting a release in the next few weeks.

A few more details on the product blog:
https://smilecdr.com/blog/20170208_custom_search_parameters
