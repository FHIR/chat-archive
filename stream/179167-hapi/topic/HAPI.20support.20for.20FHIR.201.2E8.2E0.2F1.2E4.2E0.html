James Agnew (Dec 11 2016 at 22:53):

Hi All,

FYI a new set of snapshot builds of HAPI FHIR version 2.2-SNAPSHOT have been pushed to the Maven repos. In these builds:

  • hapi-fhir-structures-dstu3 has been updated to the FHIR 1.8.0 definitions
  • Some new modules called hapi-fhir-structures-dstu2.1 and hapi-fhir-validation-resources-dstu2.1 have been added which now bring in native support for the FHIR 1.4.0 (Montreal) release.

Our public server will be upgraded to use these new definitions within the next 12 hours, and I expect to push a final HAPI 2.2 release within the next few days.
