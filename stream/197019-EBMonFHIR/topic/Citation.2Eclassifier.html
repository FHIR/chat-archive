I am wondering what the Citation.classifier is. Apparently the description of this element does not match the values in the binding value set.

Brian Alper (Jan 20 2021 at 19:14):

The Citation.classifier element is used for any number of classification tags. Many classification schemes can be applied using the CodeableConcept datatype. The description "Used for many classifiers including PublicationType, CitationSubset, MeshHeading, Chemical" is intended to convey 4 example classification schemes, each of which could have their own code systems. The binding provided as "Citation Classifier (Extensible)" is a limited set suggesting a preference to use these terms specifically when classifying a citation as one of these concepts. Perhaps we should make the binding Example instead of Extensible.
