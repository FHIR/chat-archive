## May 2019 Ballot Comment: Request event type "switch-organization"

Submitted by Ron Parker on behalf of Kenn Sinn
HL7 Canada (ken.sinn@ehealthontario.on.ca )
Chapter/section: Event Catalog
Url: http://fhircast.org/specification/May2019Ballot/#event-catalog
Type: A-S Enhancement
In Person requested: Yes :bust_in_silhouette:

Summary: Request event type "switch-organization"

Comment: Request for a new event "switch-organization", which will require the user to get a new Oauth token and refresh the context. This use-case stems from federated user logins that may be registered or works at multiple hospitals (we call it the UAO - "Under Authority Of"). The change in organization will trigger a new OAuth token to to reflect the change in user's organization, and for subscribers to also refresh the context, to ensure that are accessing FHIR server content using the updated entitlement.
