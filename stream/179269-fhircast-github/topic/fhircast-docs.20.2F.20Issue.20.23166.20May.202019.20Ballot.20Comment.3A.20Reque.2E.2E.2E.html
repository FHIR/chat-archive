## May 2019 Ballot Comment: Request event type "change-parameter"

Submitted by Ron Parker on behalf of Kenn Sinn
HL7 Canada (ken.sinn@ehealthontario.on.ca )
Chapter/section: Event Catalog
Url: http://fhircast.org/specification/May2019Ballot/#event-catalog
Type: A-S Enhancement
In Person requested: Yes :bust_in_silhouette:

Summary: Request event type "change-parameter"

Comment: Request for a generic "change-parameter" event to support FHIRcast publication of a Parameters FHIR resource. The Parameters resource will support change of search parameters, for example date range or filter parameters. This allows the subscribers to remain synchronized against the same timeframe, e.g. a Medications app, an Imaging app, an Immunizations app, for the patient-in-context, from June 01 2017 to September 01 2017. This Parameter resource could also be used to convey other elements/data for synchronization across subscribers, beyond search parameters.
