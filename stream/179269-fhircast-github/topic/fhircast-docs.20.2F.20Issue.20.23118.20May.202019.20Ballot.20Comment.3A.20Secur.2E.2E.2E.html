## May 2019 Ballot Comment: Security is not optional. Is that really required from a spec point of view?

Submitted by Ricardo Quintano Neira on behalf of @bvdh
Chapter/section: Overview
Url: https://fhircast.hl7.org/specification/May2019Ballot/index.html
Type: A-Q

Summary: Security is not optional. Is that really required from a spec point of view?

Comment: "In all cases the app authenticates to the Hub with an OAuth 2.0 bearer token.".

So, security is not optional. Is that really required from a spec point of view?

Existing wording: In all cases the app authenticates to the Hub with an OAuth 2.0 bearer token.
