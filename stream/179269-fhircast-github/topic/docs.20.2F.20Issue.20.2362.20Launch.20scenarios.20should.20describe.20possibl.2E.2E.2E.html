Github Notifications (FHIRcast) (Jan 22 2019 at 17:27):

isaacvetter opened Issue #62

During the January 2019 HL7 FHIR Connectathon our FHIRcast track discussed different app launch and app synchronization scenarios for non-SMART apps. In practice, the install base for RIS, PACS, dictation systems, etc which currently use a context synchronization mechanism in production may not support SMART launches.

We should update the informative Launch Scenarios page to better describe and recommend launch scenarios and "session discovery" not only the existing SMART flows, but also:
1. non-OAuth app launches
2. session discovery without an app launch process.

Very related to @jeremyricharson's #4 .
