## May 2019 Ballot Comment: Provide example topics to support drug and medical device research.

Submitted by Mitra Rocca; Terrie Reed
Chapter/section:
Url: https://fhircast.hl7.org/#event-catalog
Type: A-S

Summary: Provide example topics to support drug and medical device research.

Comment: It wold be useful to create example hub topics to support events based on safety for outcome research related to the use registries. For instance if a medication or device is selected in the "driving app" (e.g. an EHR) the registry/"synchronized app" could show associated safety or recall events associated with the selected "Medication" or "Device".
