Github Notifications (FHIRcast) (May 07 2019 at 04:14):

isaacvetter commented on Issue #168

Hey Kenn,

The opensource libraries at https://github.com/fhircast are implementation materials outside of the normative-track content and are intended to be supplemental material that is not part of the specification and is not being balloted.

That said, we believe that developer-friendly materials and resources, like a sandbox, a reference implementation and opensource libraries are vitally important to enabling a standards-based project. For FHIRcast, these materials are entirely volunteer-contributed. There's no funding here.

Please start by submitting bug reports on the appropriate repo? We also welcome PRs for bug fixes.

Isaac

Proposed Resolution: Not Related
