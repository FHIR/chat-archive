Summary:

Comment: One again: the second sentence asy the fields within context are subsets of FHIR resources. Then it sasy for open-image-study it should send the ImagingStudy FHIR resource.

Existing wording: FHIR is the interoperable data model used by FHIRcast. The fields within context are subsets of FHIR resources.
