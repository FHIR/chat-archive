Ward Weistra (Feb 17 2022 at 11:23):

@Brett Esler An enhancement for the Simplifier.net's package importer behind the Registry to support these old values is in the works.
The "version": "2.1.0", in the package.json of hl7.fhir.au.pd#2.2.0 will however still prevent the import into the Registry, since it can't be sure what the version is you intended for this package.
As far as the Registry is concerned you can still change the package's contents under the same 2.2.0 version number, since it hasn't been imported yet. (No need to release the updated file as 2.2.1 at this point)
