Richard Townley-O'Neill (Apr 04 2017 at 05:35):

Maybe the #Codes is meant to guide you to the _Value domain attributes_ section.
_Person—Indigenous status, code N_ is a Data Element, a combination of a Data Element Concept (Person—Indigenous status) and a value domain (code N)
So http://meteor.aihw.gov.au/content/index.phtml/itemId/602543 is for the data element and http://meteor.aihw.gov.au/content/index.phtml/itemId/602543#Codes is for the description of the codes used in that data element.
So the semantics can be different, even if the anchor has been missed.

Richard Townley-O'Neill (Apr 04 2017 at 05:36):

That would suggest checking with METeOR whether my analysis is correct, and if it is, asking them to put in the missing anchor.
