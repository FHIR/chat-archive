Becky Gradl (Jun 24 2020 at 16:46):

I don't know the history on the creation of NutritionOrder, but @Margaret Dittloff may be able to shed some light on that. The issue of using NutritionOrder in an outpatient setting has been raised in the past, however since it was not an urgent issue, no work has been done to see if it would work. We would need to do a walk-through of the resource to see if in its current state it might work in an outpatient setting for a "nutrition prescription" (i.e. a suggested diet to follow) and determine if it needs any modifications or a new resource. In this instance, I would think the diet (that would typically be used for a diet order) could be used for the "nutrition prescription". There is a bi-weekly Nutrition call on Wednesdays under O&O where we discuss nutrition topics. This would be the best place to have this conversation.
