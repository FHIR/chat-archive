Topic: Practitioner services in Catalog

Alexander Ivanov (Oct 29 2018 at 12:00):

New FHIR R4 introduces several new resources to implement a service catalog. It seems to cover lab diagnostics quite well but I have some questions about services performed by a practitioner.

I need the ability to link practitioner services (procedure codes) in the service catalog of organization with practitioners which can perform these procedures. We have some ideas but none of them seems to fit this task:
1. The catalog is the profile above composition and composition have field subject. We could have the full catalog with subject reference to organization and subsets of this catalog with the subject reference to practitioner role. But the subject field is restricted in catalog profile, so we dropped this idea.
2. We can reference HealthcareService from a CatalogEntry but HealthcareService is no longer a definition from a workflow point of view (http://build.fhir.org/workflow.html). And we think that each orderable position of a catalog should be a definition to fit workflow pattern.

So is there any other approach to link practitioner services (procedure codes) in the catalog with practitioners?
