Topic: v3.6.0 Deployed with support for FHIR v3.2.0

Richard Ettema (Jan 15 2018 at 13:30):

Touchstone v3.6.0 is now available with support for FHIR R4 Candidate v3.2.0 and the HL7 FHIR Connectathon 17 event. New test groups for FHIR v3.2.0 are now available:

  • FHIR3-2-0-Advanced - API testing using the Patient resource type
  • FHIR3-2-0-Basic - basic workflow testing by resource type (create, read, update, history, vread, search and delete)
  • FHIR3-2-0-Connectathon17 - various test tracks

Richard Ettema (Jan 15 2018 at 13:30):

Please see the Release Notes for more information - https://touchstone.aegis.net/touchstone/TouchstoneReleaseNotes
