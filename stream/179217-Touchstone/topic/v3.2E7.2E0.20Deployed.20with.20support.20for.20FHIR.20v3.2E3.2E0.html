Topic: v3.7.0 Deployed with support for FHIR v3.3.0

Richard Ettema (Apr 30 2018 at 19:06):

Touchstone v3.7.0 is now available with support for FHIR R4 Ballot 1 v3.3.0 and the HL7 FHIR Connectathon 18 event. New test groups for FHIR v3.3.0 are now available:

  • FHIR3-3-0-Advanced - API testing using the Patient resource type
  • FHIR3-3-0-Basic - basic workflow testing by resource type (create, read, update, history, vread, search and delete)
  • FHIR3-3-0-Connectathon18 - initial Test Track coverage for Cologne, Germany May 2018: Patient, Terminology and Version API

Please see the Release Notes for more information - https://touchstone.aegis.net/touchstone/TouchstoneReleaseNotes
