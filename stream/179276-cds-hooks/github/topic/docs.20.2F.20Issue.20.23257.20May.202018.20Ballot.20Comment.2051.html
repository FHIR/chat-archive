## May 2018 Ballot Comment 51

Submitted by @bvdh from Philips Healthcare

Chapter: CDS Services
Section: CDS Service Response - Card Attributes
Type: NEG :exclamation:
In Person Requested? Yes :bust_in_silhouette:

Existing Wording:

Allows a service to suggest a set of changes in the context of the current activity (e.g. changing the dose of the medication currently being prescribed, for the medication-prescribe activity). If used, the user MUST BE allowed to choose no more than one suggestion.

Comment:
How to deal with different services that address the same resource or are dependent on the same resource(s) for the calculation?

## Triage Information

Triage Notes:
Reviewed with Bas

Proposed Disposition: Persuasive with Mod
Proposed Disposition Comment:
How conflicting or overlapping decision support from CDS Services should be handled in the EHR is outside of the specification.

We will add clarifying documentation noting that the specification does not address these types of scenarios and that both EHRs and CDS Services need to consider this in their integrations.
