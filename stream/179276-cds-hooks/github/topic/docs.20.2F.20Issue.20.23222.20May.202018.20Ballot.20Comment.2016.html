## May 2018 Ballot Comment 16

Submitted by @bvdh from Philips Healthcare

Chapter: CDS Services
Section: Providing FHIR Resources to a CDS Service
Type: A-T :pencil2:
In Person Requested? Yes :bust_in_silhouette:

Proposed Wording:

Two optional enhancements are provided. First, FHIR resources may be obtained by passing "prefetched" data from the EHR to the CDS Service in the service call. FHIR resources requested in the CDS Service description are passed as key-value pairs, with each key matching a key described in the CDS Service description, and each value being a FHIR resource. For data to be prefetched, the CDS Service registers a set of "prefetch templates" with the EHR, as described in the Prefetch Template section below.
