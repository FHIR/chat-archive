## May 2018 Ballot Comment 1

Submitted by @bvdh from Philips Healthcare

Chapter: CDS Services
Section: Overview
Type: A-S :bulb:
In Person Requested? Yes :bust_in_silhouette:

Comment:
The specification does not include a list of VERSIONED references. This allows the specification to change when any of the refernces change which should not be allowed.

## Triage Information

Triage Notes:
Reviewed with Bas

Proposed Disposition: Persuasive
Proposed Disposition Comment:
We agree that when references are used, the specific version of the reference should be stated.

Eg, will add footnotes for every reference to an external document (RFCs, JSON, etc). We will get help from Bas on this change.

I think linking to the root url for the FHIR links is appropriate throughout.
