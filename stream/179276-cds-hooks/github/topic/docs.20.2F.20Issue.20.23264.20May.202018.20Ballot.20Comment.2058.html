## Triage Information

Triage Notes:
Reviewed with Bas

Proposed Disposition: Persuasive with Mod
Proposed Disposition Comment:
We will update this to clarify that update actions modify the given FHIR resource, whether that applies to a FHIR resource provided as part of the hook context or any FHIR resource available from the given FHIR server.

We will also remove references to "current activity" as is misleading and not accurate in all scenarios.

We will also ensure that all references to CDS Hooks fields are using the code notation (backticks in Markdown).
