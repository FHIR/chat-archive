## May 2018 Ballot Comment 41

Submitted by @bvdh from Philips Healthcare

Chapter: CDS Services
Section: FHIR Resource Access
Type: A-Q :question:
In Person Requested? Yes :bust_in_silhouette:

Existing Wording:

The CDS Service is able to use the EHR's FHIR server to obtain any FHIR resources it requires beyond those provided by the EHR as prefetched data.

Comment:
This is optional, only when FHIR URL is provided in calling CDS Service request.

## Triage Information

Triage Notes:
Reviewed with Bas

Proposed Disposition: Persuasive with Mod
Proposed Disposition Comment:
We will update the "FHIR Resource Access" section to clarify that if a fhirAuthorization field is sent by the EHR, it can be used to access data from the given FHIR server.
