## May 2018 Ballot Comment 34

Submitted by @bvdh from Philips Healthcare

Chapter: CDS Services
Section: Prefetch template
Type: A-S :bulb:
In Person Requested? Yes :bust_in_silhouette:

Existing Wording:

Similarly, if the EHR encounters an error while prefetching any data, the prefetch template key SHALL NOT be sent to the CDS Service.

Proposed Wording:

Similarly, if the EHR encounters an error while prefetching any data, the prefetch template key SHOULD SHALL NOT be sent to the CDS Service.

Comment:
In the case of an error, the data should not be provided as one does not know what data is correct and whether the data is still the most current one.

## Triage Information

Triage Notes:
Reviewed with Bas

Proposed Disposition: Persuasive with Mod
Proposed Disposition Comment:
We will update this from SHOULD NOT to SHALL NOT
