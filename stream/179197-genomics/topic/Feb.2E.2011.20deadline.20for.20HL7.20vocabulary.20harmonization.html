Stream: genomics

Topic: Feb. 11 deadline for HL7 vocabulary harmonization

Joel Schneider (Jan 31 2019 at 05:00):

The March HL7 harmonization meeting has a February 11 deadline for proposal submissions.

http://www.hl7.org/events/harmonization/

The following wiki page describes the harmonization process. (This document hasn't yet been migrated to Confluence.)

http://wiki.hl7.org/index.php?title=Vocabulary_and_RIM_Harmonization_Process

My understanding is that various genomics code systems may need to go through harmonization in order to be properly integrated with HL7 standards. Harmonization involves all HL7 standards, not only FHIR. Third-party code systems may additionally need to be considered by the HL7 Terminology Authority (HTA), but should go through harmonization first.

The CG workgroup has been compiling information about relevant genomics code systems in the following Google doc.

https://docs.google.com/spreadsheets/d/14sIVVED2rI0Ab2ZQ3oV6v8njzX-sRezkIWcgkkepUUg/edit#gid=0

If possible by Feb. 11, I think it would be best to identify genomics code systems we are ready to advance through the harmonization process, and construct a harmonization proposal for those.

This topic was briefly discussed during last Tuesday's CG call, where we decided to continue discussion here, in Zulip.
