Larry Babb (Sep 20 2019 at 19:13):

Just sharing that the GENO ontology is now providing well vetted terminology for several of the LOINC codes we previous enumerated answer lists for, which might be better served in the long run by having a reference to an ontological identifier with a much more robust way to express the answers at varying levels of granularity. Matt Brush from Monarch Initiative at OHSU worked on this with others including the ClinGen experts to develop these terms that fall outside of Sequence Ontology yet are pretty essential to this space. They certainly filled some critical gaps for defining genotypes and inheritance patterns.

Here's the mapping of these value sets with those we defined in ClinGen to inform the GENO work (along with a note on a forthcoming Chromosome ontology to provide chromosome identifiers for all organisms - seems like a smart approach for a standard to consider).

Phase Set .

ClinGen allelic phase value set maps to GENO terms here

Allelic State

ClinGen allelic state value set maps to a subset of the GENO terms here
