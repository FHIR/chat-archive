Topic: Fix for Extension Validation

Grahame Grieve (Dec 18 2019 at 22:33):

I am releasing a new validator/publisher shortly that properly validates cross version extensions - see http://hl7.org/fhir/versions.html#extensions

A side discovery while implementing this - The FHIR Validator has been ignoring extensions that weren't identified with proper URLs. Not any more... it now flags these as invalid and checks the inner non-absolute URLs (they must be defined by the extension definition or they are invalid)

If this creates new errors in your IGs that are up for publishing, we will not hold them up for this. But you should fix any errors that arise if you can
