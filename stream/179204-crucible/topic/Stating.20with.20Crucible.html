Jason Walonoski (Apr 23 2018 at 14:40):

If you are using the public instance, enter your publicly accessible URL into the "FHIR Server URL" field and click the "Begin" button. From there you select which tests you want to run and then click "Execute".

Jason Walonoski (Apr 23 2018 at 14:41):

If you are trying to setup your own private instance, you can follow the installation instructions on the wiki: https://github.com/fhir-crucible/crucible/wiki
