Hola!! Acá les dejo como realizar la validación contra un profile:

1. Crear profiling deseado en Forge
2. Asegurarse que En "Options" se encuentre seleccionado "Save Snapshop Component"
3. En pestaña "Properties", cambiar en valor de la última parte del campo "URL" "http://example.org/fhir/StructureDefinition/[CambiarEsteTexto]", este corresponde a nombre del profiling construido.
4. En Forge, click botón derecho sobre nombre del profile creado --> Publish to FHIR server --> [Escribir dirección del servidor] --> Connect --> Save.
5. En pestaña "Properties" --> "Resorce ID", puedes ver el "id" del recurso "StructureDefinition" que contiene al profile generado por si quieren revisarlo (La modificación realizada se encuentra bajo el tag <differential>).
6. En cliente REST realizan un POST a [url-FHIR-Server]Patient/$validate
7. En el cuerpo del mensaje copian el recurso que quieren validar contra el nuevo profile, para ello se especifica sentro del campo "meta":
{
"meta":{"profile"="http://example.org/fhir/StructureDefinition/[CambiarEsteTexto]"},
"resourceType": "Patient",
"identifier" : {
"value" : "15945528-2"
},
......
}
8. Ejecutan y el servidor les retormará los comentarios respectivos..
