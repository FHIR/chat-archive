Robert Dieterle (Mar 25 2021 at 16:56):

The VHDir was an international standard for directory information exchange. The work on this stream is focused on the FAST Directory solution implementation guides. The FHIR IG proposal (in development) relevant to VHDIR will use the learning from the VHDIR and add 1) updates developed for PDex Plan Net to support US realm requirements (e.g. code systems and value sets), 2) specific extensions (e.g. to the endpoint resource to support additional metadata) to support the issues identified by FAST, and 3) any additional requirements identified by the work at HL7 under the PA workgroup.
