Marco Visser (Sep 24 2020 at 10:21):

At Firely we released the 2nd beta release for fhir-net-api version 2.0. The official version is 2.0.0-beta2 .

In this release you will find also support for R5. We use the preview #3 Fhir specification (http://hl7.org/fhir/2020Sep/).

The final release of 2.0 is almost there. We would love to receive your experiences of upgrading the fhir-net-api in your own projects to this new version. You can let us know by creating an issue on GitHub when encountering problems in the upgrade.
When the all clear is given, we will ship version 2.0 ASAP.

See here the announcement made by Ewout Kramer and here the document about breaking changes for 2.0 (because there are some in this release).

This is a beta2 version, so not production ready yet!

See here the releases on GitHub:

  • STU3: https://github.com/FirelyTeam/fhir-net-api/releases/tag/v2.0.0-beta2-stu3
  • R4: https://github.com/FirelyTeam/fhir-net-api/releases/tag/v2.0.0-beta2-r4
  • R5: https://github.com/FirelyTeam/fhir-net-api/releases/tag/v2.0.0-beta2-r5

The packages can be found on NuGet: https://www.nuget.org/packages?q=hl7.fhir

The Firely team
