Q1. A state should have one FFS PlannetInsurancePlan instance. In addition, one PlannetNetwork instance would be created for each network of providers and organizations within a state’s FFS Medicaid program. The PlannetInsurancePlan would have a reference to each network of providers participating in that plan (for different benefits and geographic coverage areas, etc.).
Q2. The Medicaid Agency would be identified through the InsurancePlan resource. The PlannetInsurancePlan.OwnedBy field would reference the Medicaid Agency Organization Resource (e.g. ‘Virginia Medicaid Program’). The PlannetInsurancePlan.Type field would be set to ‘Medicaid’
