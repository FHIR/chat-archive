Ward Weistra (Apr 23 2020 at 16:31):

Hi @Mareike Przysucha, I just looked into this and the project is depending an older version of the Simplifier.Core.STU3 package which didn't include the profile http://hl7.org/fhir/StructureDefinition/shareablevalueset.
Newer versions of that package do include it. However, it would be even better to update the dependency of the project to the official package hl7.fhir.r3.core now that's available (see our latest blog).
