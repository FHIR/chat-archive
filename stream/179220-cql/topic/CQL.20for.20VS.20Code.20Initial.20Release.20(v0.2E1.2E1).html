Topic: CQL for VS Code Initial Release (v0.1.1)

JP (Sep 24 2021 at 21:03):

We've published an initial preview release for Clinical Quality Language (CQL) for VS Code, for those of you that use VS Code instead of Atom. It should have feature parity with the Atom plugin, including:

  • CQL syntax highlighting
  • CQL error highlighting
  • Local CQL execution
  • Viewing ELM

Right now it's a direct port of Atom. We expect it to evolve to be more VS Code friendly as we go along.

cqframework.cql

You can install it by searching for "CQL" in the VS Code Extension panel.

Please report any issues here
