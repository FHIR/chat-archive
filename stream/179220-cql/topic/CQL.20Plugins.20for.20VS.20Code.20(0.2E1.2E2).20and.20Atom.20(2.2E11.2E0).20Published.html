Topic: CQL Plugins for VS Code (0.1.2) and Atom (2.11.0) Published

JP (Jan 03 2022 at 00:05):

We have released new CQL Plugins for VS Code and for Atom. These releases include:

New Features:

  • Support for the cql-options.json file in an IG
  • Translator option and language level appropriate error messages (as opposed to default options)
  • Support for executing CQL versions 1.3-1.5 (previously it was 1.5 only)

Updated Dependencies

  • Updated cql language server to 1.5.3
  • Updated to cql-translator 1.5.6
  • Updated to cql-evaluator 1.4.1

These should be automatically installed next time you open your editor:
https://atom.io/packages/language-cql
https://marketplace.visualstudio.com/items?itemName=cqframework.cql

Please report issues to the appropriate repository:
https://github.com/cqframework/atom_cql_support/issues
https://github.com/cqframework/vscode-cql/issues
