Chris Moesel (Dec 17 2021 at 21:01):

Hi @Tien Thai. The only required environment variable now is UMLS_API_KEY -- as it replaces UMLS_USER_NAME and UMLS_PASSWORD. Please ensure that your environment variable is set in such a way that it is available to other processes. How you do this depends on your operating system, but... are you setting it in the same terminal/console that you're using to launch CQL Services? If you set it in a different terminal tab or window, then it might not be available to the CQL Services process. Also ensure you're using the latest version of CQL Services (v2.0.0).
