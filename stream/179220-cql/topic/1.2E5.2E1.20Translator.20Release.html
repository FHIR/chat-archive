Topic: 1.5.1 Translator Release

Bryn Rhodes (Dec 08 2020 at 18:51):

We have released the 1.5 CQL-to-ELM translator and associated tooling. The release packages have been uploaded to Maven Central.

Release notes are available here:

https://github.com/cqframework/clinical_quality_language/releases/tag/v1.5.1

This release implements full support for the normative aspects of the 1.5 specification, and partial support, as detailed in the release notes, for the trial-use aspects. Focus for the next release is on:

  • Profile-informed model info processing and mapping
  • Enhanced model info capabilities
  • Improved support for include/reverseInclude and searchPath in retrieves
