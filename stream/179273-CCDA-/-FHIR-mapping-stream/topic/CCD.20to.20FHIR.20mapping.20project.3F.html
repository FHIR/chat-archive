Lloyd McKenzie (Jul 13 2019 at 22:21):

V3 RIM mappings exist, but their purpose is definitional. Both CDA and the RIM allow the same information to be expressed in too many (unpredictable) ways to perform generic transformations. Any such mappings need to be done to specific v3 specifications (e.g. CCDA). Even CCD is too generic/flexible to produce reliable mappings.

Lloyd McKenzie (Jul 16 2019 at 15:51):

The hierarchy is RIM -> CDA -> CCD -> CCDA with the RIM being most abstract. C-CDA is mostly tight enough to perform transformable mappings. For higher level specifications, you'll need a tighter implementation guide if you want to perform transformation. Every implementation guide will need its own set of mappings/transforms.
