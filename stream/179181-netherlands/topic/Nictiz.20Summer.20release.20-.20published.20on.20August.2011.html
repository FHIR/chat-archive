Hi all,

Last week, Nictiz has pre-released the upcoming Summer release, containing breaking, backwards imcompabible changes for FHIR conformance resources and MedMij information standards used in the Netherlands.

I would like to encourage all of you to take a look at the intended changes, please feel free to comment and/or create a pull request. This is especially helpful if you are already depending on the current Nictiz package and/or using the Dutch profiles.

The pre-release can be found on GitHub and the packages on Simplifier.
