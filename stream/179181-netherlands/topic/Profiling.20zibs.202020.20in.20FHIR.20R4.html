Topic: Profiling zibs 2020 in FHIR R4

Ardon Toonstra (May 27 2021 at 11:39):

We have started profiling zibs from release 2020 in FHIR R4. The profiles are developed and maintained in the 'Nictiz-R4-zib2020' repository.

Encountered mapping issues are documented and handled with the GitHub repository issues. Everyone is invited to review, think along and hopefully provide solutions! We welcome all your feedback, here on Zulip or in the issues.

Occasionally, we may place relevant issues as separate topics here on zulip to attract more attention. Updates on the status of the profiling work will be added in this topic.

Ardon Toonstra (Oct 04 2021 at 14:39):

Here's an update!
We now have the Nictiz R4 Zib2020 Simplifier project which contains conformance resources for zibs that are ready to be tested in a proof of concept, kickstart, trial implementation or of course by you :wink: ! For this purpose, a package (beta versioned) is created: nictiz.fhir.nl.r4.zib2020.

Expect more conformance resources (and fixes/improvements) to be added soon once they are through the entire development pipeline.

Let us know if you have any feedback, preferably through the use of the GitHub repository issues.
