Alexander Henket (Dec 15 2017 at 09:31):

In case you want to search for more stuff in the OID Registries we have: http://decor.nictiz.nl/decor/services/OIDIndex

Any OID can be exported as NamingSystem. I import the HL7 International OID Registry regularly and combine that with the FHIR NamingSystems. The HL7 OID Registry contains the leading registration data and the FHIR NamingSystems tell me if there is a URI for an OID and if that URI is preferred within FHIR (normally is).
