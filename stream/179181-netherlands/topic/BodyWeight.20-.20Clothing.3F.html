Ardon Toonstra (Aug 19 2021 at 11:09):

Hi @Bert Verdonck , welcome here ;)
Please keep in mind that when implementing you would best navigate through the implementation guide in order to get to the correct profile. So not sure which use case you are implementing, but if it is for example BgZ, this would be the implementation guide which reference the correct profiles in the table.
Some bulletwise observations:

  • You refer to the BodyWeight zib in release 2020. No published profiles exists for them. They are currently being developed.
  • The FHIR profile is a copy of the HL7 FHIR vital signs profiles and fixes some errors that have been fixed in R4 already. It is not the 'end' profile. Which would be: https://simplifier.net/packages/nictiz.fhir.nl.stu3.zib2017/2.1.5/files/389765
  • There is a Observation.component slice in that profile that holds the clothing concept of the zib.
