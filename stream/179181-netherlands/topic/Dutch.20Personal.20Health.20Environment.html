For the Dutch Personal Health Environment (draft English name) (in Dutch 'Persoonlijk Gezondheids Omgeving') @Alexander Henket @Arianne van de Wetering and I are creating FHIR Profiles that might be helpful for the entire Dutch Realm (and perhaps even internationally). We are using the basic health data set (Basis Gegevenset Zorg) as the basis for the project, and we try to limit project specific profiles as much as reasonably possible.

We will probably come at a point where we are creating conformance resources which would be better to be part of the Basic Components. ( https://simplifier.net/NL-BasicComponents ). When we arrive at this point we will discuss this with all of you if this is the case.
