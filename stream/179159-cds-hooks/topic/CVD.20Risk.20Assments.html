Kevin Shekleton (Dec 13 2016 at 22:01):

I've thought quite a bit about a CVD risk assessment CDS Hooks card and provided all of the data needed for the calculation were available via the FHIR server, you could display a simplified view fo the score in the card with a link to a SMART app to view a more rich view of that same data.

Kevin Shekleton (Dec 13 2016 at 22:02):

But, if there is any missing information in FHIR that is necessary for the calculation, the only thing you could do is return a card that links to the SMART app where the user could interact with the UI to enter in the missing data (this is what the current Cardiac Risk SMART app allows for today)
