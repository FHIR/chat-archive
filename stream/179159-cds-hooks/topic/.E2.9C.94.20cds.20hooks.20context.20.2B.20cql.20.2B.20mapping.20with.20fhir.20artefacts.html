Thanks Bryn for the link, very interesting and helpful. An other question please: is it possible to access cds hooks context from a cql code using parameter, someting like that:

Bryn Rhodes (Feb 09 2022 at 16:10):

That mapping right now isn't very sophisticated, but it will pick up a single list-valued parameter in the CQL and bind that to the hook context. I submitted an issue to enhance that binding support to bind by context parameters from the hook by name to parameters in the CQL. https://github.com/DBCG/org-opencds-cqf-cds/issues/41
