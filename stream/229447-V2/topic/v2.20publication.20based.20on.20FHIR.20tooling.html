We are working oin several things in parallel: fixing bugs in v2.x, technical terminology, integration of all parts, tool stack, persistence layer, structure and layout. What you can see is something I generate with scripts from my database. With that procedure we are collecting requirements to get it done. The primary challenge is making it persisting with FHIR resources so that the whole process can be done with FHIR tools. That would be something where FHIR can take advantage from.
