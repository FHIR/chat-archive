Topic: Annotation on a list of medications

Richard Townley-O'Neill (Nov 26 2019 at 06:48):

We have a requirement to indicate that a list of current medications includes some (unidentified) ones that are included in a dose administration aid ("A sealed, tamper-evident device that allows individual medicine doses to be organised according to the prescribed dose schedule.")
This is NOT information about any one item in the list.
We're looking at an extension on List of type CodeableConcept with a ValueSet and CodeSystem with values like "Some items included", No items included", "Unknown".
Does anyone currently record this another way?
