Topic: More than one NDC pointed to the same RXCUI

Courtney Bland (Apr 07 2022 at 19:20):

On the formularies that we share with CMS, we have instances of more than one drug that possess the same RXCUI, but are treated as different drugs. We currently have RXCUI as part of the primary key for our Medication Knowledge/Formulary Drug records so a member won’t get back more than one record with the same RXCUI. We were hoping to get some advice on how we should handle this scenario.
RXCUI: 309686
Display Name: Dexamethasone (Oral Liquid)
Full Name: dexamethasone 0.1 MG/ML Oral Solution
NDC 60432046608 is for Dexamethasone 0.5 mg / 5 mL Elixir Bottle 8 oz.
NDC 00054317763 is for Dexamethasone 0.5 mg / 5 mL Solution Bottle 500 mL
Both NDCs share the same RXCUI of 309686, according to the NIH, but they are considered different drugs on our formulary because one is an Elixir and one is a Solution. In many cases, multiple NDCs pointing to the same RXCUI is not an issue because they really are the same drug, but we do have these instances in which they are different.
