Where the QHP formulary has an enumerated list with etc at the end, we have a value set with an extensible binding. For example drug_tier. Extensible means you can use something else if your concept isn't covered in the list provided.

network_tier is defined in the QHP documentation for use with providers, so does not appear in the formulary IG.
