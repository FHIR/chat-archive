Bryn Rhodes (Feb 03 2021 at 21:34):

The CQL-based HQMF IG (and soon the QM IG) specify how to calculate opportunity scoring measures from the component measures. There is no need to specify additional calculation semantics within the measure itself. In the same way that a proportion measure doesn't need to specify how to calculate the performance rate.
