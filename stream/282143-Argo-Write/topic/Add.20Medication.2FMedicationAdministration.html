I came across the document at https://hackmd.io/@erichaas/rJVqJGmeY/%2FwTGb4Gk6R6O4NVut5yaJig. Not sure if it still current, but I'd have a comment.

In my use case, remote monitoring of type 1 diabetes, most data is represented as Observations, but at least we (Sensotrend) are using MedicationAdministration for insulin doses from insulin pumps and "smart" insulin pens that track doses automatically. Very much patient generated data, and a very common use case.

I'd think the same is true for smart inhalers, for instance.

So I propose to add Medication/MedicationAdministration on the Medications row. Or at least consider this detail for the resulting IG.
