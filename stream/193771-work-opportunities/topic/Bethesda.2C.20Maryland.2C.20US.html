SOFTWARE DEVELOPER

The LHNCBC requires the services of a software developer who will be responsible for supporting the development of tools for facilitating the moving of standardized clinical data for use in clinical care and research. The core of this project is LHC-Forms, a form-rendering widget for medical forms, which imports and exports data in the HL7 FHIR format. There are also a growing list of associated tools with some connection to FHIR, including a clinical table web API service, a flowsheet, a form builder, and units of measure tools. These tools are mostly written in JavaScript (both client and server code) with a little Java. The position reports to the Acting Chief, Computer Science Branch (CSB), and has the following duties:

  • Full-stack web application development.
  • Eventually managing a substantial module of the project.

This opportunity is full-time with MSC and it is on-site in Bethesda, Maryland. https://careers-mscweb.icims.com/jobs/3580/senior-web-application-developer-in-a-red-hat-environment/job Medical Science & Computing is an Equal Opportunity/Affirmative Action employer. All qualified applicants will receive consideration for employment without regard to race, color, religion, sex, national origin, disability, or protected Veteran status.
