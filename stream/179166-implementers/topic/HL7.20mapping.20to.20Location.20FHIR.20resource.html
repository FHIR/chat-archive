Craig Newman (Jul 31 2020 at 13:52):

I don't think I've ever seen the LOC segment used. The V2-to-FHIR project does have a mapping for the PL data type to Location (although it gets a bit complicate because it creates a series of Locations for the bed, room, facility, etc). Check out https://docs.google.com/spreadsheets/d/17b4wqJOzGieNBSdHYZt3ymKucsRDd_h8leaiz_umG7E/edit#gid=0 and let us know if you have thoughts on the mapping

Cooper Thompson (Jul 31 2020 at 14:43):

I second Craig's comments - I've only ever seen location in HL7v2 be sent in the context of something else (e.g. PV1, FT1, etc.) using the PL data type.
