Michel Rutten (Nov 01 2019 at 12:46):

Note: Forge does allow you to slice the Extension.value[x] element and define a type choice - should you need this. FHIR also allows you to further constrain the default extension slice discriminator (by adding additional discriminators), however this advanced technique is currently not supported by Forge.
