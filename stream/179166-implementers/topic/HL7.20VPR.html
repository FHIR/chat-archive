Bryn Rhodes (Feb 01 2017 at 15:35):

Yes, Virtual Medical Record was designed to be used as the payload for Clinical Decision Support. The requirements for vMR were used as input for the QI-Core profiles. The HL7 CDS WG maintains the vMR standard, and it can be used with the HL7 Knowledge Artifact Specification to describe decision support artifacts. There are also template specifications which define bindings for the vMR for use with decision support. The logical model for vMR is defined here: http://www.hl7.org/implement/standards/product_brief.cfm?product_id=338, vMR templates are here: http://www.hl7.org/implement/standards/product_brief.cfm?product_id=339, and the XML representation can be found here: http://www.hl7.org/implement/standards/product_brief.cfm?product_id=342.
