John Moehrke (Oct 06 2020 at 12:52):

as Dave points out, the question is too open-ended... That said, IHE has defined an Document Sharing HIE that uses a set of FHIR Resources. This is outlined in the Mobile Health Document Sharing (MHDS) implementation guide.
https://wiki.ihe.net/index.php/Mobile_Health_Document_Sharing_(MHDS)

Derek Ritz (Oct 06 2020 at 15:22):

There's a community of practice that has defined an HIE architecture based on IHE Profiles: OpenHIE (https://ohie.org/#arch). The latest version of this architecture is based on FHIR (and heavily references the MHDS profile).
