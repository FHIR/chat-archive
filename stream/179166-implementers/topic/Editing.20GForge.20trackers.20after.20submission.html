I've never been able to edit my GForge issues after I submit them either. The best I've been able to do is add a comment indicated what should be changed in the issue. It would be nice if we were allowed to edit our own issues.

Lloyd McKenzie (Feb 01 2019 at 15:48):

Only "managers" can edit tracker items. You can however add a comment indicating your intent and someone else can make the change for you. (In practice, targetRelease is only supposed to be specified once the change is approved, so capturing it in a comment for now is appropriate.)
