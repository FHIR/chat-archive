Topic: Value Set Defining URL

AbdulMalik Shakir (Mar 06 2019 at 01:12):

I have broken links in my IG when using the defining URL for a code system or value set. For example code system List Order Codes has a defining URL of http://hl7.org/fhir/list-order. When traversing the link I get a 404 page not found. However, when I navigate to the code system I land on page https://hl7.org/fhir/2018Jan/codesystem-list-order.html. Which URL should I use in my IG?

Richard Townley-O'Neill (Mar 06 2019 at 01:25):

Try http://terminology.hl7.org/CodeSystem/list-order

Lloyd McKenzie (Mar 06 2019 at 03:36):

Richard is correct. You use the URL defined for the code system in the value set. Whether it resolves is irrelevant. You can see the proper URL in the spec here: http://build.fhir.org/valueset-list-order.html
