Topic: What Identifier Type to use for pseudonyms

Noemi Deppenwiese (May 28 2020 at 13:01):

In a research setting, all our patients are assigned pseudonym identifiers instead of their original patient ids from the hospital system. While the original IDs have an Identifier type (https://www.hl7.org/fhir/valueset-identifier-type.html) of MR, what is the correct code for a pseuonymised ID? Is it still a MR? Or a filler or placer identifer?

Vassil Peytchev (May 28 2020 at 13:46):

It sounds like creating and using a new type of "pseudonym" might be the best solution.
@Rik Smithies @Hugh Glover has this come up in BRR?

Lloyd McKenzie (May 28 2020 at 14:14):

It's still MR. You can use another code too, but with an extensible binding you must use the codes from the value set if they apply - even if they don't convey the level of detail desired.
