I am having an issue with the CQL plugin for Atom where it claims that it "could not to load source library" in my include statements. It then proceeds to show an error for anything that references that library.

Winston Leung (Sep 16 2021 at 15:33):

I've reinstalled the package half a dozen times, but finally managed to resolve the issue by carrying out the installation in a very particular manner. Probably exists some leeway around the steps, but this was how I got it working on Atom and was able to replicate it on Atom Nightly, which also had the same issue.

  1. Uninstall atom-ide-ui then language-cql.
  2. Restart Atom.
  3. Install language-cql.
  4. Open a folder, you'll be prompted to install the atom-ide-ui dependency. Click Yes.
  5. After that has been installed, open a .cql file.
  6. The cql language server will automatically install. I haven't had installation fail on me with this procedure, but if it does I'd suggest restarting from the beginning.
