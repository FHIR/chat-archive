In the description of the who and onBehalfOf fields in Provenance there is a comment saying "whoIdentity should be used when the agent is not a Resource type.". I cannot find whoIdentity anywhere. What is whoIdentity?

John Moehrke (Dec 04 2020 at 17:02):

very good question. can you submit a change request for me to fix that. It should say "who.Identity" to indicate that the Reference datatype .identity element can be used when there is not Reference.reference. This might not be needed to be said anymore now that this concept is more clear in the datatype Reference.
