Sean McIlvenna (Jan 19 2022 at 19:41):

@John Moehrke has submitted a JIRA ticket suggesting that we consider deprecating DocumentManifest in time. Structured Documents WG is interested in knowing if there are any implementers using DocumentManifest today. Originally, DocumentManifest was created to address IHE scenarios. IHE is no longer using DocumentManifest. We're looking for any other use-cases beyond IHE. Please let me know in this thread. Much appreciated!

Sean McIlvenna (Jan 19 2022 at 19:41):

Here's John's JIRA ticket in case anyone is interested: https://jira.hl7.org/browse/FHIR-33021

John Moehrke (Jan 19 2022 at 20:00):

to be clear. Without others using DocumentManifest, it was obvious that IHE alone would not be able to compel DocumentManifest to mature to Normative. So IHE looked at use of List, and found it useful. So we now use List where we used DocumentManifest before. We also use List for the XDS SubmissionSet, in addition to Folder now.
