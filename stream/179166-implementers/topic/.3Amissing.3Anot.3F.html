Topic: :missing:not?

Paul Lynch (Sep 17 2021 at 21:20):

Is there some way of querying whether there is any data for a given search parameter to find? For example, for the Observation category search parameter, I would like to find out whether there are any Observations that have any category value set. It occurred to me that category:missing:not=true would be what I wanted, but typically :missing is not supported and I don't think you can combine modifiers that way.

Michele Mottini (Sep 17 2021 at 21:29):

Wouldn't that be category:missing=false ?

Paul Lynch (Sep 17 2021 at 21:33):

Oh, yes. You are right. Unfortunately, I still have the problem that :missing is not usually supported. There are performance issues with indexing records that don't have something. I would think that problem wouldn't apply for :missing=false. Maybe just that half of :missing could be supported. If not, there some other way to query for that?

Michele Mottini (Sep 17 2021 at 23:03):

No other way I know of. (And yes, it is not widely available - our server supports :missing but only for reference search parameters)
