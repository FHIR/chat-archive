Topic: Forge 16.1 for STU3

Michel Rutten (Jun 16 2017 at 13:41):

Forge 16.1 for STU3 is now available for download from https://simplifier.net/forge.

This new version represents a minor update. It contains fixes for a couple of issues that were reported by the community since the initial STU3 release for the HL7 WGM in Madrid. Please note that not all known issues have been fixed yet, as we did not want to delay publication of the update any longer. We'll try to address the remaining known issues in a future release.

Mr. Forge is now taking two weeks vacation and will be back online in July. If you find any new issues, please report them at forge@furore.com.

Happy profiling!

Michel Rutten
