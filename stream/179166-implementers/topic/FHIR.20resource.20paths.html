Pascal Pfiffner (Oct 31 2016 at 15:46):

Note that http://hl7.org/fhir is not actually a FHIR server, it hosts the documentation. If you want to download the sample resources, you can find a link to archives for those in XML, JSON or RDF format here: http://build.fhir.org/downloads.html
