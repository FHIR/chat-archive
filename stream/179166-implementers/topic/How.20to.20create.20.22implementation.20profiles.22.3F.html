Topic: How to create "implementation profiles"?

John Silva (Jan 23 2018 at 02:51):

In HL7 V2 there were mechanisms and tools to create "implementation profiles"; does FHIR have a similar mechanism and tooling? I found this webpage (maybe dated) that talks about spreadsheet-based profiling; is this the way it's done today or is there better, easier to use tooling available? http://wiki.hl7.org/index.php?title=FHIR_Spreadsheet_Profile_Authoring

Lloyd McKenzie (Jan 23 2018 at 04:03):

For profile authoring you can use spreadsheets, Forge, Trifolia or direct XML or JSON creation. Profiles can be published as part of implementation guides using the HL7 IGPublisher or Simplifier.

John Silva (Jan 24 2018 at 11:03):

Thanks. I tried Forge and it's pretty easy to use. My question about it (and other FHIR profile tools) is: "Is there a way to view these profiles without using an online website like Simplifier.net?" In other words, is there a stylesheet or similar for displaying these 'locally' or a tool that creates a document (e.g. PDF) from these profiles?

Grahame Grieve (Jan 24 2018 at 11:13):

the IGPublisher can do that. You just point it at the profile(s) and it will publish a plain representation of them
