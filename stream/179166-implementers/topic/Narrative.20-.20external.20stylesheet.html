Topic: Narrative - external stylesheet

Shamil Nizamov (Aug 08 2018 at 21:00):

One more possible inconsistency: in 2.4.0 Narrative "The XHTML content SHALL NOT contain … external stylesheet references …"
Later there, 2.4.0.4 Styling the XHTML "The XHTML fragment in the narrative may be styled using cascading stylesheets with either external or internal styles. External styles are applied using the class and id attributes on the XHTML elements ..."

If a reference is not allowed how do class and id can apply external CSS?

Grahame Grieve (Aug 08 2018 at 21:04):

In order to minimize manageability and security issues, authoring systems cannot specify the CSS stylesheet to use directly. Instead, the application that displays the resource provides the stylesheets.
