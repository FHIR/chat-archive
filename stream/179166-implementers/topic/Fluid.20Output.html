Robert McClure (Aug 10 2017 at 16:20):

Ok, based on a discussion with Clem, it seems representing the duration of collection as a UCUM unit is not acceptable. The expectation is that each drain type collection duration would be a unique LOINC code (i.e.; it is a specific test, and likely would be ordered that way.) There quite a few in there already, just not for the ones you want. So I think you need to request them as new LOINC codes using the approach noted here https://loinc.org/submissions/new-terms/
