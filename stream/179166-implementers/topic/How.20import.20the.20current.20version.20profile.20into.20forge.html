Michel Rutten (Sep 28 2017 at 08:50):

Hi @Fan Lin, Forge is designed to author FHIR profiles. You can create a new profile using the File/New Profile command. First you need to select the base resource or datatype that you want to profile, e.g. the Patient resource. Forge will then create a new profile on the selected base. Now you can start applying constraints.
