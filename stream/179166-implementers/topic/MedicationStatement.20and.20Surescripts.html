Surescripts doesn't support a FHIR interface for ePrescribing in general. We are using FHIR for a few products in specific ways, but it is not possible to do something like send a prescription using FHIR at the moment. Surescripts using the NCPDP SCRIPT standard to do that. Hope that helps.
