Brian Postlethwaite (Sep 23 2016 at 19:07):

summary is explicit, only those marked with the summary flag should be included, and the returned resource should be marked with the subsetted tag. (as noted here)
http://hl7.org/fhir/2016Sep/search.html#summary
Invariants don't matter here, and edits really shouldn't be done on subsetted content, as posting it back as an update would wipe out any values in non summary properties.

Brian Postlethwaite (Sep 23 2016 at 19:10):

As for the summary=count, this just means that the entries collection is not returned.
The Bundle.Total property contains a value that represents the number of the resource you are searching for, and excludes any _include or _revinclude resources. As per spec here:
http://hl7.org/fhir/2016Sep/search.html#count
