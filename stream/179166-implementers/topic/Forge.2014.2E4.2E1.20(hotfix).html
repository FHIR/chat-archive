Topic: Forge 14.4.1 (hotfix)

Michel Rutten (Dec 05 2016 at 14:54):

Forge hotfix release 14.4.1 is available for download from http://www.simplifier.net/
This release represents a minor update with a couple of bug fixes.
Users of Forge 14.4 will receive an auto-update notification.
Users of earlier release must download and install the update manually.

We've also published two blog articles about the new Forge release.
The first article is intended for Forge end users and provides a high-level overview of important new features: https://thefhirplace.com/2016/12/05/forging-profiles-on-profiles/
A second article is targeting a developer audience and describes some of the technical challenges that we faced: https://thefhirplace.com/2016/12/05/forge-14-4-technical-overview/

If you would like to submit a comment, suggestion or bug, we welcome your feedback at forge@furore.com.

Happy profiling!
