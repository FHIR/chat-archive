Topic: Creating a ValueSet for certain MIME-types in STU3

Morten Ernebjerg (Jan 22 2020 at 15:44):

I am trying to define a ValueSet containing a specified set of MIME-Types in STU3. For R4, there is a MIME-Type ValueSet that gives the coding system as urn:ietf:bcp:13. However, for STU3, the binding I have seen just refers to the set of all MIME-types by giving the URL to the RFC (https://www.rfc-editor.org/bcp/bcp13.txt) and does not specify a coding system. What is the correct way of defining a ValueSet sub-set in this case? -If I read the spec right, there is no way to sub-set a ValueSet by listing specific codes without giving a system, but urn:ietf:bcp:13 only seems to turn up in connection with R4.
