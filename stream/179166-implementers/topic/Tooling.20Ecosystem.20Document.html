Lloyd McKenzie (Sep 10 2017 at 23:58):

To start the process of getting greater interoperability between FHIR tooling and to focus attention on the areas that most need work, we've started a wiki page that describes the tooling ecosystem. The page is here: http://wiki.hl7.org/index.php?title=FHIR_Tooling_Eco-system

The content there is not complete. We're probably missing tooling areas and we're definitely missing some of the tooling implementations. As well, the notions about architecture are a straw model for discussion - we need agreement from the community that this is what we want to do and how we want to do it. So if you're involved in creating tooling or interested in using tooling that supports FHIR development, please review and provide feedback. Discussion will happen on the tooling stream and potentially on some FHIR-I calls. If there's sufficient volume of feedback, we may schedule some tooling-specific calls.
