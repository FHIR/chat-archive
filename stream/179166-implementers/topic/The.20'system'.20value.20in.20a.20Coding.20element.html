Topic: The 'system' value in a Coding element

Brian Reinhold (Mar 12 2018 at 13:41):

Is the intent that the coding.system value should always be a functional 'link' to something or is it, in general, a string that uniquely identifies the coding system?

Rob Hausam (Mar 12 2018 at 14:50):

It's in general a URI that does the latter - uniquely identifies the code system. But, whenever possible, it should be a resolvable URL, and ideally it should point to the authoritative source of the code system (a CodeSystem resource, if possible) from the code system publisher.
