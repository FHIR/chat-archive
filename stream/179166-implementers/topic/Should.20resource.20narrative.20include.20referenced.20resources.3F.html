Topic: Should resource narrative include referenced resources?

Tim Berezny (Feb 28 2018 at 16:04):

If i'm sending a /ServiceRequest, in the narrative should it include details ONLY in the serviceRequest fields, or can/should it include details of the references resources (like, /patient name and address?)

Lloyd McKenzie (Feb 28 2018 at 16:43):

It can include a subset of information in referenced resources, but typically that will be pretty minimal.

Lloyd McKenzie (Feb 28 2018 at 16:44):

(The challenge being that the referenced resource might evolve separately)

Lloyd McKenzie (Feb 28 2018 at 16:44):

If the reference is to a contained resource, the narrative should include all relevant information
