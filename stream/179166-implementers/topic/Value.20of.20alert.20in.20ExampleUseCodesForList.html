Please could a colleague clarify for me the intended meaning of the code "alert" in the ValueSet with url http://hl7.org/fhir/ValueSet/list-example-codes. I would like to use it to indicate an alert that the patient record for a specified patient has been accessed by an IT system or person using an IT system. Hence my List's entries will be AuditEvents on a specified patient's record. Is that a possible usage for this code?

Lloyd McKenzie (Mar 16 2021 at 13:52):

An 'alerts' list would typically be a list of Flag resources. If your intention is to have a List that highlights data accesses, I'd probably define a code specifically for that. The list-example-codes is just an example binding - there's no expectation that the list is complete and no requirement to use the codes provided
