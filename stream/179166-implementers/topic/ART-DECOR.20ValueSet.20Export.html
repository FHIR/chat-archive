Alexander Henket (Sep 17 2019 at 18:01):

Hi. You are trying to retrieve a value set from a project demo1- that does not have that value set. Hence you get an HTTP 404.

If you want to retrieve data from a specific project, you should use the project endpoint. If you don't you should use the public endpoint.

The value set you seek may be retrieved from:

http://art-decor.org/fhir/3.0/public/ValueSet/1.3.6.1.4.1.19376.1.3.11.36--20151113191623

Alexander Henket (Sep 17 2019 at 18:04):

More documentation is on the art-decor.org documentation wiki: https://art-decor.org/mediawiki/index.php?title=FHIR_URIs

Alexander Henket (Sep 17 2019 at 18:08):

Another hint for finding the right links: go to your ART-DECOR project page. If your project does not yet mark FHIR as a relevant standard (restURI listing), please add it there.
Then click the ProjectIndex link. Links for datasets/transaction/value sets are in that overview and includes the FHIR links for any version your project chooses to support.
