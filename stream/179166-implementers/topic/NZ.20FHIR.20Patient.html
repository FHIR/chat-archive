Not yet - for R3 or R4 anyway - there are some issues to be resolved around Code Systems and Value Sets. However, the bigger picture is that NZ now has a central repository of patient demographic data (NHI) that's updated dynamically by point-of-care systems, so the need for healthcare facilities to exchange much of this data between themselves has been much reduced.
