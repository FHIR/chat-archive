Topic: Validator Bug

Grahame Grieve (May 28 2019 at 21:05):

I just found and fixed a serious issue in the validator: under some circumstances, the validator would fail to check value set membership for some coded values. Specifically, when validating against an IG that restricts the value set of a FHIR defined code system, the code would be accepted if was a valid code, not if it was valid according to the IG.

  • Resources that may have been deemed to be valid in the past may now fail.
