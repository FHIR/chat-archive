Brian Postlethwaite (Jul 26 2017 at 05:25):

base/Patient/45 = 410 (gone) for deleted resource
base/Patient/45/_history (might = 404 if server doesn't support versioning, but will return a bundle that has the last entry as the deleted one in cases where versioning is supported)
base/Patient/45/_history/v3 = 410 (gone) for a versioned request to the deleted version
base/Patient/45/_history/v2 (will return the requested version you asked for)

Brian Postlethwaite (Jul 26 2017 at 05:28):
