Lloyd McKenzie (Feb 08 2021 at 23:06):

There generally aren't sample "messages" for FHIR because the FHIR core specification doesn't define any standard message definitions. (Where messaging is used, it leverages message definitions developed for use in specific communities and documented in implementation guides specific to those communities.) However, you'll see several examples of Location instances on the example tab for the Location resource - http://hl7.org/fhir/location-examples.html
