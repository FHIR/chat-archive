Topic: HTML in "String" dataType?

Tim Berezny (Mar 10 2018 at 19:28):

Is it generally considered ok to include basic HTML formatting in elements of type annotation or string (such as serviceRequest.note serviceRequest.PatientInstruction)?

Lloyd McKenzie (Mar 10 2018 at 19:45):

Only if your users like looking at HTML tags :) If you want a string to be rendered, send the unformatted string and also send a http://hl7.org/fhir/extension-rendering-xhtml.html. If the receiver supports the extension, they'll render the "pretty" version. If not, the user will at least get the information - with no offending tags in the way.
