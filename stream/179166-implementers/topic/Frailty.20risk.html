Giorgio Cangioli (Mar 14 2019 at 15:43):

I've been asked to represent a Frailty risk . The result of this evaluation is not a percentage, but a score.
My understanding is that the appropriate resource for this is in any case the RiskAssessment using the qualitativeRisk to record the score and the basis for referring the input evidences. But I appreciate if you can confirm this, or if you suggest to use an observation for this scope.

Giorgio Cangioli (Mar 14 2019 at 15:53):

Associated question: in case this information would be collected into a section do you suggest to use for this section the 71482-4 LOINC code (Risk assessment Document) or the 75492-9 Risk assessment and screening note LOINC code ? (or others)

Lloyd McKenzie (Mar 14 2019 at 15:56):

RiskAssessment is essentially a specialized form of Observation. If all you need is a single value, Observation is fine. RiskAssessment is about a more complex pattern of expression of risk over time.
