Topic: Does ProcedureRequest.code allow two code systems?

Sylvia Freeman (Aug 22 2018 at 23:34):

Hello, I wanted to know if there is a way to set up the DSTU2 ProcedureRequest profile so that the code element can take CPT (http://www.ama-assn.org/go/cpt) or LOINC (http://loinc.org)? I am using Forge 18.2.

Lloyd McKenzie (Aug 23 2018 at 16:15):

There's no ability to specify multiple bindings for ProcedureRequest.code. However you can slice ProcedureRequest.code.coding and provide separate bindings for different slices.

Sylvia Freeman (Aug 23 2018 at 17:42):

Ok, thank you Lloyd!
