What is the application of the following statement from the narrative spec (https://www.hl7.org/fhir/narrative.html) to the XML special characters (&,<,>,",') contained within a text.div?

"Note that the XHTML is contained in general XML so there is no support for HTML entities like &nbsp; or &copy; etc. Unicode characters SHALL be used instead. Unicode &#160; substitutes for &nbsp;."

For example, if I am adding "History & Physical" to a text div, should this be encoded as "History &#38; Physical", or, "History &amp; Physical".
