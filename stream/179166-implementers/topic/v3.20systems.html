Rob Hausam (Aug 26 2019 at 14:20):

They are both code system urls, as you say - the first is the url in STU3 and the second is the url in R4. Normally the canonical urls for code systems don't change, but in this case it was felt that the urls or a whole set of HL7 defined and managed code systems needed to be brought in alignment under terminology.hl7.org as we move toward the unified UTG terminology infrastructure (and content when possible) across all of the HL7 standard families (V2, V3/CDA and FHIR). In R4 you use http://terminology.hl7.org/CodeSystem/v3-ParticipationType for 'system'. And in STU3 you still use http://hl7.org/fhir/v3/ParticipationType (I just verified that on tx.fhir.org).
