Stefan Lang (Jun 22 2018 at 15:51):

Click on the "Extension" root element and use the "Add" button (top left) to add an element.

Stefan Lang (Jun 22 2018 at 15:51):

Or use the context menu of the "Extension" root element

Stefan Lang (Jun 22 2018 at 15:54):

You can also add to the child elements to create nested hierarchies, if required
