Cesar Farell (Aug 02 2017 at 18:25):

Is it legitimate to invoke an operation that can be invoked on an instance on a specific version? E.g. is this legal: [base]/ValueSet/[id]/_version/[version]$expand ?

Consider that the $expand operation has an optional date input parameter for filtering to which version of the ValueSet the operation should be applied. Having both a date and a version together would likely produce an overdetermined and incoherent request: what if the requested version was not effective at the specified date?
