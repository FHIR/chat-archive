Hi @Ramachandran G, FHIR STU3 has introduced breaking changes with respect to DSTU2, so the API releases for DSTU2 and STU3 are not compatible.

Michele Mottini (Sep 09 2017 at 23:02):

Michele Mottini (Sep 09 2017 at 23:03):

new Hl7.Fhir.Model.Extension
            {
                Url = "http://hl7.org/fhir/us/core/StructureDefinition/us-core-race",
                Extension = new List<Hl7.Fhir.Model.Extension>
                {
                    new Hl7.Fhir.Model.Extension( "ombCategory", new Hl7.Fhir.Model.Coding( "urn:oid:2.16.840.1.113883.6.238", "1000-9", "White" ) ),
                    new Hl7.Fhir.Model.Extension( "text", new Hl7.Fhir.Model.FhirString( "White" ) ),
                    new Hl7.Fhir.Model.Extension( "detailed", new Hl7.Fhir.Model.Coding( "urn:oid:2.16.840.1.113883.6.238", "2108-9", "White-European" ) )
                }
            };
