Grahame Grieve (May 24 2018 at 17:51):

conditional update:
- if you don't find a match, create
- if you find a match, update it
- if you find multiple matches

Grahame Grieve (May 24 2018 at 17:51):

conditional patch:
- if you don't find a match, ignore it
- if you find a match, update it
- if you find multiple matches
