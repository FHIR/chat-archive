Brian Postlethwaite (Jul 19 2021 at 23:22):

There is also the enableBehaviour property that indicates if all or any of the enableWhen options are selected, you could could create something like this:

"enableWhen": [
  {
    "operator": "=",
    "question": "Q11",
    "answerString": "option1"
  },
  {
    "operator": "=",
    "question": "Q11",
    "answerString": "option2"
  }
],
"enableBehaviour": "all"

This would only then activate when Q11 has both option1 and option2 selected.
or it you had any instead of all, it would be an or style test.
