Lloyd McKenzie (Sep 24 2018 at 14:52):

General summary is that there's no requirement to declare a version other than in the server's CapabilityStatement. Requesting a specific version might be by URL (if a server has different version-specific endpoints) or via the Accept header (if a server supports that).
