Bob Milius (Aug 08 2017 at 20:44):

Does anyone have any examples of using CDEs from the cancer Data Standard Repository (caDSR) in a codeable concept? I found an example online in Questionnaireresponse-sdc-profile-example.xml for an old version of the FHIR specification, but it doesn't look like it made it to the current version. Wondering if there was something 'wrong' with it?

Lloyd McKenzie (Aug 09 2017 at 02:02):

It was moved into an independently published implementation guide. The current version is here: http://hl7.org/fhir/us/sdc/index.html
However, that particular example was just a proof-of-concept, so definitely don't take it as gospel. Certainly you can treat a repository of CDEs as a code system if you define a URL for the namespace of "all CDE identifiers from repository X".
