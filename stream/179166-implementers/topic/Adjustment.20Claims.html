Looking for recommendations from anyone who has processed adjustment claim records including void records. Can you describe how you represented them in FHIR? For void records, did you make fields negative so that the fields negate the original claim? Example fields would be all financials, quantity fields, and days. For financial records, do you anticipate that only the financial fields would be filled out on the record but fields for quality fields and days have a value of “0”? For reversals (cancellations) you don't send an empty or whatever claim, you send a cancellation, see the http://hl7.org/fhir/financial-module.html page and see Task for an example.
