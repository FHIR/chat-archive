Brian Postlethwaite (Sep 30 2016 at 04:21):

FhirClient client = new FhirClient("[your endpoint]");
client.OnBeforeRequest += (object sender, BeforeRequestEventArgs e) =>
{
  // Replace with a valid bearer token for this server
  e.RawRequest.Headers.Add("Authorization", "Bearer XXXXXXX");
};

Or you can do whatever else you need to do with the headers ;)
