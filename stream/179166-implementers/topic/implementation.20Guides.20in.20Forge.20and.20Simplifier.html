Michel Rutten (Aug 09 2017 at 15:41):

Hi @Brian Reinhold, correct, IG support is preliminary and currently under active development.

Forge allows you to author a FHIR IG resource. This was mainly implemented to allow users to open/save a set of related resources as an IG package. You cannot author actual IG content pages from within Forge.

Simplifier currently provides an online IG editor (in beta). The editor allows you to author and publish actual page content and embed profile renderings. We are currently working on improving this functionality, e.g. allowing you to import/export actual FHIR IG resources and markdown pages. However these new features are currently being developed and not yet available. We're planning to start releasing new IG features on Simplifier later this summer. Stay tuned!
