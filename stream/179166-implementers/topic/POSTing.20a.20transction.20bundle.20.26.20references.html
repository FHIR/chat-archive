Michele Mottini (Jun 11 2019 at 23:44):

No, the bundle entry containing the Patient should have a fullrUrl xxxxx and the secondary resource reference contains the same xxxxx
(where xxxxx is some valid URI unique within the bundle - eg urn:uuid:<GUID>)
