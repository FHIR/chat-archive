Lloyd McKenzie (Oct 14 2016 at 03:49):

It's been discussed. The general perspective is that Questionnaire is for raw capture, but not for retrieval/analysis. General expectation is that if you want to analysis/filtering/query, you migrate the data into the appropriate resource type (Observation, MedicationStatement, FamilyHistory, whatever). Querying by questions is actually quite tricky because you may care about the context of other questions to filter and it gets ugly pretty fast. You can always define custom search criteria if you really need the functionality
