Jessica Bota (Sep 24 2020 at 20:08):

The Unified Terminology Governance (UTG) project has updated the current build content at build.fhir.org/ig/HL7/UTG/ to include information and direct links to educational material in the Documentation tab. Please make sure you are accessing UTG from the current build as Version 1.0.0 (terminology.hl7.org) was a prototype to debug the content, process and tooling. The content there is insufficient and should be consumed with that in mind.

Currently the UTG Jira project is focused on recording and shepherding HL7 terminology content changes through consensus review. Its use will be extended to include the workflow to record and resolve requests for changes to the supporting content on terminology.hl7.org.

I apologize if you have seen this message on other streams. It has been requested in each location for transparency.
