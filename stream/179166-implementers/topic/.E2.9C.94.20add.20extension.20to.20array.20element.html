Topic: ✔ add extension to array element

Yunwei Wang (Oct 27 2021 at 13:46):

I want to use DAR extension on a Patient.name element which is an array type so I could specify why patient does not have any name. What the JSON should look like?
Looks like this is fine with FHIR validator.

  "name": [{
    "extension":[
      {
        "url": "http://hl7.org/fhir/StructureDefinition/data-absent-reason",
        "valueCode": "not-asked"
      }
    ]
  }],

Is this the correct way to use DAR on array type?

Lloyd McKenzie (Oct 27 2021 at 14:20):

Technically, that's a reason why this one name is missing, not why the collection is missing. There's no way to put an extension on the collection as a whole.
