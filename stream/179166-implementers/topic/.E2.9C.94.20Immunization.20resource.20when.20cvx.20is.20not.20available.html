Topic: ✔ Immunization resource when cvx is not available

Gayathri Narayanan (Jul 30 2021 at 06:06):

There are scenarios when a vaccine will not have cvx value. This cvx value is required to populate the mandatory field - vaccineCode (https://www.hl7.org/fhir/us/core/StructureDefinition-us-core-immunization-definitions.html#Immunization.vaccineCode).
How should we display these types of vaccines which don't have a cvx code ?

Cooper Thompson (Jul 30 2021 at 12:57):

You can DAR it (Data Absent Reason), as described in Missing Data.

Lloyd McKenzie (Jul 30 2021 at 13:56):

I don't think you can use DAR and still communicate an alternate code?

Daniel Venton (Jul 30 2021 at 14:15):

You don't have to use cvx, if there is no cvx. The binding is extensible and it even says that in your link, "The codes SHALL be taken from CVX Vaccines Administered Vaccine Set; other codes may be used where these codes are not suitable". If you have an alternate, put the alternate. If you don't know anything then do you really have record of an immunization? If you only have text, the perhaps the DAR as the code with the text = "Gorilla Flu Vaccine" might be the best you can do.
