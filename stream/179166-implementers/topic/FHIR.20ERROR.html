The following code works for me:

IGenericClient client = fhirCtx.newRestfulGenericClient("http://hapi.fhir.org/baseDstu3");
Patient myPatient = client.read().resource(Patient.class).withId("1326789").execute();
System.out.println("Family name: " + myPatient.getNameFirstRep().getFamily());
