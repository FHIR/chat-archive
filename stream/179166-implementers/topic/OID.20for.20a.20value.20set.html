Eric Haas (Aug 03 2017 at 14:18):

Rob Hausam (Aug 03 2017 at 14:46):

You don't necessarily need an OID to use the value set in FHIR. The ValueSet.identifier element is optional, but can be used to provide an OID reference, if needed. If you do need an OID and don't have another obvious way to obtain one within your particular context, then I would suggest considering registering it in the HL7 OID Registry: http://www.hl7.org/oid/index.cfm?ref=common
