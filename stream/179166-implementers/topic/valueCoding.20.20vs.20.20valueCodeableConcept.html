What's the difference between valueCoding vs valueCodeableConcept? They appear to be both under the Coding structure.

Lloyd McKenzie (Jan 09 2019 at 21:59):

CodeableConcept supports multiple translations - and is what most FHIR elements uses. For example, it would allow you to send a local code, a SNOMED code and an ICD10 code for a diagnosis.
