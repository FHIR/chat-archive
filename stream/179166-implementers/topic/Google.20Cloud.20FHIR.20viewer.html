Topic: Google Cloud FHIR viewer

Paul Church (Mar 17 2021 at 17:39):

The Google Cloud Healthcare API now has a FHIR viewer built in to the cloud console UI allowing users to browse, search, and navigate through FHIR resources stored in the Healthcare API. It displays resources with an overview tab, a navigable view of all elements, and the JSON representation, and it supports R4, STU3, and DSTU2.
