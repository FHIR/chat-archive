Melva Peters (Aug 06 2018 at 14:54):

It looks like it includes the following values: ‘0’ – Not Specified
‘1’ – Not a Compound ‘2’ – Compound and is part of the claim segment (at least that's what I see when I google it). There is nothing like this in the medicationDispense - we have a code if there is one and if not, the medication is a reference to a medication resource (either stand alone or contained). @Shalinee Batri you can add a GForge Tracker item and the Pharmacy Work Group will discuss.
