Mirjam Baltus (Nov 10 2016 at 11:38):

You'll need to select your profile or extension in your session. Then use File --> Publish to FHIR Server and enter your server endpoint. When you click connect, the server will be asked for its Conformance statement, so Forge can check to see if you can upload to it.
