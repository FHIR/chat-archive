Topic: Change to value set and code system URLS

Grahame Grieve (Jul 23 2018 at 22:18):

Notice for all implementers: one of the bigger changes in R4 will be a sweeping change to terminology canonical urls - that is, specifically, CodeSystem and ValueSet URLs. Many URLs are changing to be based on http://terminology.hl7.org instead of hl7.org/fhir). The v2 and v3 content has already changed in the draft at build.fhir.org, and I will be migrating any value set or code system where the system URL appears in instances (e.g. Coding/CodeableConcept bindings) to terminology.hl7.org as well.
