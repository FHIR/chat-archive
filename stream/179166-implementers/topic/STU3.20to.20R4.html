Lloyd McKenzie (Mar 05 2020 at 16:48):

The Java reference implementation includes migration code for converting structure definitions, but that doesn't change the elements of the profile, just the syntax they're represented in. I'm not aware of anything that does the profiles themselves. Typically you need a human in that loop because changes to data types, cardinalities, addition/removal of elements, etc. means you need to think about what the rules should be for the new profile
