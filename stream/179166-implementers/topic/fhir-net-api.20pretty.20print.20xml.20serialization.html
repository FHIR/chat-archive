Hi @Sean McIlvenna

The way I do this is as follows:

First, serialise to XML via the API
1) String cc = FhirSerializer.SerializeToXml(_resource);

Then use LINQ to format it
2) var dd = System.Xml.Linq.XElement.Parse(cc).ToString();

That works for me
