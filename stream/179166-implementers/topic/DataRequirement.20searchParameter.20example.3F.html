Bryn Rhodes (Oct 01 2020 at 21:31):

The searchParameter element is the name of the search parameter to be used for the code filter, the value is specified using the code element:

"codeFIlter": {
  "searchParameter": "status",
  "code": [{ "code": "confirmed" }]
}
