John Moehrke (Sep 24 2021 at 13:03):

FYI: The Implementation Guide specification for the API to MHDS is ... MHD... and available https://profiles.ihe.net/ITI/MHD/
The MHDS is just a specification that adds a central FHIR Server to host all the server side stuff, and bring in other IHE specifications for security, privacy, user, patient management, configuration management, vocabulary management, and time -- MHDS is not published in IG format - https://www.ihe.net/uploadedFiles/Documents/ITI/IHE_ITI_Suppl_MHDS.pdf
