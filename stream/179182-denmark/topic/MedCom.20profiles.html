René Spronk (Jan 17 2020 at 11:00):

It all depends on how one wishes to do the profiling - profiling a complex structure using StructureDefinitions only is possible, as long as one regards the message structure as a tree with outbound references between the nodes in the tree. GraphDef allows one to define the overall structure of the message. You'd do so in conjunction with StrucDefs that each describe the constraints on a node in the graph.
