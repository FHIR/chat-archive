Grahame Grieve (Sep 30 2020 at 06:24):

We continue to have problems with realms for IGs. We need a formal check list that we follow when we approve proposals for IGs, and for publication approvals.

Proposals for IGs:

  • Check that the IG realm matches the Project realm

Requests to Approve publications

  • recheck the realm
  • check that the project sponsors have approved the publication request

note that I found a project today where the PSS and the project definition differed :-(, and where a co-chair of one of the sponsors claimed that the sponsor committee hadn't approved the publication request, though I read the record differently.

Anyhow, doing this is a big value proposition - we have a demonstrated ability to get this wrong consistently as a community
