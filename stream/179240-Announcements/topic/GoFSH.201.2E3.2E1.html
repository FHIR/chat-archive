Topic: GoFSH 1.3.1

Julian Carter (Sep 08 2021 at 18:51):

Introducing GoFSH 1.3.1! GoFSH 1.3.1 contains the following enhancements and bug fixes:

  • Optimizations to shorten the length of GoFSH's processing time
  • Fixes an issue where GoFSH would log errors when encountering a JSON with a leading BOM
  • Fixes an issue where GoFSH would import a Profile of a Logical Model as an Instance
    See the release notes for additional details.
