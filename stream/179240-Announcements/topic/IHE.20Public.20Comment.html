Steven Nichols (Sep 02 2020 at 19:01):

IHE IT Infrastructure is seeking comment on a white paper that proposes a content profile for cataloging system configuration properties.

Configuration management of system-to-system interfaces that enable IHE profiles is burdensome within the healthcare enterprise. Survey of Network Interfaces Form, (SNIF) https://ihe.github.io/SNIF/SNIF-Whitepaper.html provides a consistent method to catalogue configuration properties for human and machine consumption during installation, upgrade and repair.

This 3 minute video describes SNIF https://youtu.be/EbA3tdycTqs.

We would appreciate any comments in GitHub https://github.com/IHE/SNIF/issues and particularly looking for assistance in answering open questions and issues https://github.com/IHE/SNIF/blob/master/Issues.md

Steve
