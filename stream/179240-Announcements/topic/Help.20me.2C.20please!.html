@Husayn Hasanov Please do not ask questions on the #Announcements stream. Implementation questions should be raised on the #implementers stream. Also, this forum is about asking questions about how to use the FHIR standard - it's not for asking questions about the capabilities or features of arbitrary healthcare devices. I would suggest researching the device by make & model and, if necessary, reaching out to the manufacturer. If it turns out that the device has a native FHIR interface (unlikely but possible) and you have challenges understanding how to consume/use that interface once you get it enabled, the #implementers stream is a great place to raise those questions.
