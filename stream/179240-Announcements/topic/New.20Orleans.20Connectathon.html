Grahame Grieve (Oct 04 2017 at 21:20):

Heads up to everyone: FMG has decided that in New Orleans (Jan 2018), there will be some changes to the FHIR Connectathon:
- the Connectathon will run all day on Sunday (instead of finishing at lunch time)
- We will no longer have a report back session for each stream. Streams will still report, but via some social media (TBA yet).
- we will use the break out rooms for stream demonstrations on Sunday afternoon
