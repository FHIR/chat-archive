Topic: Looking for good FHIR Test server which can handle bundle

Gary Fung (Jan 17 2017 at 20:04):

Hi there, I am new in this group. I am looking for a good FHIR Test server which can test the bundle message with type=message and Messageheader resource. I googled a bit but most of them only support bundle transaction, but not bundle.type = message. Any URl is appreciated.

Thanks

Lloyd McKenzie (Jan 17 2017 at 22:12):

You can store bundles of any type on most test servers. If you want the message actually *processed* using the $process-message endpoint, that's harder - because the responses to every single message can be different - they'd have to be coded separately. Generic servers can't reasonably be expected to do that.
