Dan Gottlieb (Jan 22 2018 at 20:56):

Hi - I just created a new zulip stream for discussing the proposed FHIR Bulk Data API (http://www.healthintersections.com.au/?p=2689) called "bulk data".

Also, the SMART team has released a reference implementation of a FHIR Bulk Data API server designed to help developers explore the proposed API. If you’re attending the Connectathon Bulk Data track this weekend, you can use it when working through the scenarios outlined at http://wiki.hl7.org/index.php?title=201801_Bulk_Data .

The code for the server is at https://github.com/smart-on-fhir/bulk-data-server , an online demo is at https://bulk-data.smarthealthit.org , and we posted a sample command line client at https://github.com/smart-on-fhir/sample-apps-stu3/tree/master/fhir-downloader .

Please post questions and comments to the new stream!
