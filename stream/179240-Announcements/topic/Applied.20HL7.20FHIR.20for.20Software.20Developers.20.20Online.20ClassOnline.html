Applied FHIR for Software Developers online class tarts on May 4th.

This online class takes an in-depth look into the infrastructure sections of the HL7® FHIR® specification. It includes slide-based tuition alternating with hands-on-sessions where participants apply what they know.

As new concepts are introduced, participants will get to use FHIR "hands-on" for what they have learned previously and for some of the more advanced concepts introduced in this course. Exercises are performed with XML or JSON tools and a REST client e.g. Postman.

Participants perform the activities themselves during the sessions with the instructor available for help and to demonstrate the solutions. Writing program code will not be required.
Register now at: http://www.hl7.org/events/webinar/fhir/developer/2021/05/index.cfm

Please contact sadhana@hl7.org if you have any questions.
