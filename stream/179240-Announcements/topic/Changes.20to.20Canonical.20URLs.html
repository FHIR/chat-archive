Topic: Changes to Canonical URLs

Grahame Grieve (Sep 18 2019 at 14:17):

the Clinical #genomics (CG) workgroup has asked Vocab to correct several canonical URLs in the terminology list:

  • http://www.ncbi.nlm.nih.gov/refseq/ -> http://www.ncbi.nlm.nih.gov/refseq
  • http://www.ncbi.nlm.nih.gov/clinvar/ -> http://www.ncbi.nlm.nih.gov/clinvar
  • http://varnomen.hgvs.org/ -> http://varnomen.hgvs.org

That is, remove the trailing slash. The trailing slashes appear to be have been introduced into the R4 spec as an editorial error.

Vocabulary WG (#terminology ) have approved making this change as part of the technical correction due out in a couple of weeks if the CG work group can provide good evidence to the Vocab committee by 27th Sept (internal deadline for the technical correction) that this change will not impact on implementers. If you are an implementer that uses one of these code systems, please contact @Patrick Werner ASAP.
