Now that we have finished the work on R4, our next project - just a small one this time - is to publish a technical correction for R3. We will be considering things that are breaking implementations that can be fixed by technical corrections (mainly clarifications, and corrections to collateral, and implementation suggestions). We won't consider anything that requires changes to confomant implementations.

So far, this is what is on our list: GF#13865
