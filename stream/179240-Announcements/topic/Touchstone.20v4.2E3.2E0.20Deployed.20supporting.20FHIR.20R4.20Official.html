Topic: Touchstone v4.3.0 Deployed supporting FHIR R4 Official

Richard Ettema (Jan 09 2019 at 18:10):

AEGIS is pleased to announce that the Touchstone Testing Platform 4.3.0 and TestScript Editor 1.3.0 have now been released. Significant milestones for this release:

  • Support for FHIR R4 Official (v4.0.0). See http://www.hl7.org/fhir/directory.cfml and http://hl7.org/fhir/R4.
    • Touchstone now offers FHIR R4 Official (v4.0.0) testing and support for the HL7 FHIR Connectathon 20 event.
    • FHIR R4 (v4.0.0) test scripts can be found at FHIR4-0-0-Connectathon20, FHIR4-0-0-Basic and FHIR4-0-0-Advanced test groups under Test Definitions.
    • CDS Hooks support in Touchstone now also supports FHIR R4 Official (v4.0.0).
  • The TestScript Editor now also supports FHIR R4 Official (v4.0.0). You can download the latest here.
  • TestScript authors can now deactivate test groups to reduce clutter on the Test Definitions tree.
  • Support for HTTP Method in Test Script operation. See TestScript.setup.action.operation.method.
  • Security and Section 508 improvements.
  • See Release notes for more details.
