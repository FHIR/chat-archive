Topic: SMART App Launch: Ballot Sign-up

Josh Mandel (Jul 28 2017 at 16:16):

For anyone interested in the SMART App Launch specification, please be aware that the ballot signup deadline is fast approaching! To sign up, you'll need to take the following steps by Thursday 8/3:

1. Visit the HL7 Ballot Desktop
2. Sign in as a member or non-member participant (note: a $400 fee applies to non-members)
3. Check the box for "HL7 FHIR® IG: SMART Application Launch Framework, Release 1 (PI ID: 1341)"
4. Continue the sign-up process to completion

If you are participating in the ballot, please consider submitting issues directly via GitHub and then referencing these issues from your official ballot submission. Also note that even if you don't participate in the official ballot, you can always provide feedback directly via GitHub.
