The latest validator from HL7 can be found here: https://github.com/hapifhir/org.hl7.fhir.core/tree/c5255a0f80c5ed683540d5809baa2341e6715b67/org.hl7.fhir.validation/src/main/java/org/hl7/fhir/validation

If you are looking to run any testing against a FHIR server, you can utilize the Touchstone FHIR Testing tool at touchstone.com, which offers profile validation as part of the testing process.
