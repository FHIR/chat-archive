David Hay (May 04 2021 at 23:29):

For trackleads - I've added all the tracks into conman - or at least the basics - just a link to the confluence site. If you are thinking of using conman, then there are a couple of extra steps:

If you are testing an IG, then you'll want to create a link from the track to the IG/s. To do this:

  • Check that the IG is in the list of IGs - add it if not
  • From the Track tab, click on the edit icon to the right of the track name, which will bring up the edit dialog. Make sure the track type is 'IG Review' and you should have a list of IGs in a dropdown. Select the one you want, then click the green 'add' button to add a link from the track.
  • you may want to add yourself as the track lead - then only you can make changes. Sorry, right now there is only a single track lead

If you want to use conman to capture feedback (and I'll be sending out more info shortly) then you'll need at least one scenario. To do this

  • select the track
  • in the description tab to the right you'll see an 'Add a new scenario' link. Clicking that brings up the scenario dialog.
  • give the scenario a name and a brief description, then Add. (You can do more, but that's probably enough for most). This will allow users to add feedback from testing to the scenario

Any issues report them here - it's great to get this stuff sorted well ahead of time if possible so we have time to resolve any issues...

David Hay (May 06 2021 at 07:26):

Also for track leads - I've just pushed an update to make it easier for users to add feedback linked to a scenario - that could be a comment or the result of a test. Theres a short video that describes this, but briefly a user can add a comment linked to a scenario and optionally specifying a server or IG. The comments can be downloaded as a csv file for later review.

Obviously, you'll need to add scenarios to your track for this to work.

You may well already have a way to capture this information, but if not then hopefully this will be of use.
