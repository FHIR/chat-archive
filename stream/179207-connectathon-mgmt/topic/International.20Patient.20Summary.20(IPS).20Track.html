Peter Williams (Feb 02 2020 at 02:57):

If it's of interest, I've put the IPS onto our development server as a ValueSet. So you could look up the members of that and - for example - filter to just the ones that contain 'hernia' using: https://dev-fhir.snomedtools.org/fhir/ValueSet/ips/$expand?filter=hernia any questions you're welcome to see me at the Terminology Services table.

Peter Williams (Feb 02 2020 at 04:15):

I was asked how that was defined internally as a FHIR Resource. We used the SNOMED ECL expression for "memberOf" a reference set. See here: https://gist.github.com/pgwilliams/f9a51007e02870b4f5f2010ad4266f31 I've done this example using the GPS because that SNOMED Identifier has been published officially.
