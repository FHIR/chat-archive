David Hay (Jan 05 2019 at 20:40):

btw - if you're unsure which tracks you wish to participate in, you can view the track details directly from conman. To do this, load the app ( http://conman.fhir.org/connectathon.html?event=sanant2019 ) then select any track. In the tabset to the right is a new tab headed 'Details Page'. In this tab is displayed the details page for the track. If you leave that tab open and select another track then the details page for that track will be displayed (after a short pause).

This is a simple way of quickly viewing all the track details

David Hay (Jan 05 2019 at 22:37):
