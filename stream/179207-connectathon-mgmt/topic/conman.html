David Hay (Sep 26 2018 at 19:16):

Hi Guys, it would be great if people attending the connectathon put their contact details in conMan (CONnectathon MANager btw). It should be simple:

1. Navigate to http://conman.fhir.org/connectathon.html?event=baltimore2018
2. At the login screen, click the 'Register new person' link, and enter your name there, then login to the app. No password is needed.
3. Find yourself in the 'peoples' tab (there's a filter box above the list to make it easy) then edit your entry to add contact details - especially your primary track and tracks of interest.

You only need to do this once - the login details are cached in the browser cache (If you use a different browser, then you'll be able to select your name from the list - you don't need to register again).

If you are hosting a server, then enter the details on the server tab (appears when any track is selected). Remember to click the 'Check CapabilityStatement' link to load the capabilityStatement which sets the FHIR version (and also shows the 'Server Capability tab'

If you want to view the capabilities of an existing server, then select the 'view' link in the list and click the 'Check CapabilityStatement' link again (the CapabilityStatements are not saved in the database as they get large so need to be loaded each time)

To see who in interested/participation in each track, use the 'Participation Report' tab.

If there are any issues - use this chat. See you in Baltimore!

David Hay (Sep 14 2019 at 20:32):

The tab only shows when a technical track is selected (I'm going to remove non-technical tracks after the event so it will always appear then.
To add a server to a track, select the track, then the 'technical testing tab. You'll see all the scenarios for that track in a list across the page. To the right of each scenario is an 'add server link that allows you to add a server to that scenario in a given role (which is defined in the scenario). After this, the track/role will appear alongside the server in the servers tab.

There's a client link to the left that does the same for clients.

User Interface (and functionality) suggestions always welcomed!
