Topic: SUSHI 2.2.6

Chris Moesel (Jan 06 2022 at 00:43):

Announcing SUSHI 2.2.6 with the following bug fixes and minor enhancements:

  • Supports in-lining FSH-defined profiles, value sets, and code systems in instances (e.g., in contained, Bundle.entry.resource)
  • Fixes bug in XHTML processing that could cause invalid XHTML output

NOTE: As of the time of this release, the security certificate on the FHIR package server is still expired. As a result, dependency downloads will fail (if you don't already have them in your cache). This affects all versions of SUSHI. Once the server certificate is updated, dependency downloads will work again. Until then, you may choose to disable SSL certificate validation (but please read and understand the cautions).

Install or Update

To install or update to this release, run the following command:

$ npm install -g fsh-sushi

More Information

For more details on this release, see the SUSHI 2.2.6 Release Notes.
