Topic: GoFSH 1.0.1

Nick Freiter (Mar 03 2021 at 20:46):

Announcing GoFSH 1.0.1 with the following enhancements and bug fixes:

  • Add support for assigning a generated id to resources which are missing an id
  • Add support for rules on a CodeSystem that include a code with spaces
  • Fix bug causing improperly escaped quotes in some circumstances
  • Fix handling of resources that have a name containing whitespace

See the GoFSH 1.0.1 Release Notes for more details.
