Topic: SUSHI 0.12.1

Nick Freiter (Apr 15 2020 at 19:06):

Announcing SUSHI 0.12.1 with the following enhancements and bug fixes:

  • Adds -f|--force flag to _updatePublisher.sh and _updatePublisher.bat to force the script to run without user input. Can be used in non-interactive settings (thanks for the contribution @Pétur Valdimarsson )
  • Adds the ability to fix inline instances to elements of any matching type, not just elements of type Resource (#347)
  • Fixes issue with setting version on compose.include elements in a ValueSet (#346)
  • Fixes issue with finding paths with matching elements on instances (#357)
  • Fixes issue with fixing values on slices that do not have associated discriminators (#362)
    Release notes: https://github.com/FHIR/sushi/releases/tag/v0.12.1
