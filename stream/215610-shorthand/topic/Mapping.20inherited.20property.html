Nick Freiter (Apr 20 2020 at 11:56):

It looks like SUSHI is failing on the id element because it does not have a mapping field already defined on it, but the name and text fields already have some existing mapping. So this looks to be a bug, I added this issue https://github.com/FHIR/sushi/issues/372 to track it.
