Topic: Announcing SUSHI 0.8.0

Chris Moesel (Feb 08 2020 at 01:16):

Announcing SUSHI 0.8.0 with the following features, enhancements, and bug fixes:

  • Supports copying and configuring multiple pages, intros, notes, and images in ig-data
    • See README for details on page support and configuration
  • Logs info messages to inform users of progress in SUSHI execution
  • Automatically fixes the url value for extensions that are defined inline (#178)
  • Fixes package resolution for current versions of packages (#201)
  • Fixes support for paths containing contentReferences (e.g., item.item on Questionnaire) (#184)
  • Fixes incorrect base differential element for profiles with no constraints (#198)

https://github.com/FHIR/sushi/releases/tag/v0.8.0
