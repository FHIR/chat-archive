Topic: FSH Finder

Max Masnick (Nov 16 2020 at 19:27):

Introducing FSH Finder :fish::telescope: (https://fshschool.org/fsh-finder/), an automatically generated list of IGs using FSH.

IGs are identified by crawling GitHub repos registered for build.fhir.org CI, and by crawling a static list of GitHub users/organizations. If you know of an IG using FSH that you think should be listed but isn't, please open an issue on GitHub. Updates will be ~weekly (until automated regeneration is set up), so if you register for build.fhir.org it may take a bit for your IG to appear here.

FSH Finder also attempts to determine how FSH is being used (defining profiles, instances, OperationDefinitions, StructureDefinitions) using the GitHub Search API, which is only available for the default branch on the repo.

Please let me know if you have any questions or run into bugs. Thanks!
