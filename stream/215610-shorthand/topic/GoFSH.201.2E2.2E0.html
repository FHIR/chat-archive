Topic: GoFSH 1.2.0

Nick Freiter (Jun 07 2021 at 19:45):

Announcing GoFSH 1.2.0. This release contains the following enhancements and bug fixes:

  • Adds support for FHIR versions above 4.0.1
  • Logical models and custom resources are now exported as instances of StructureDefinition
    • NOTE: Support for the new FSH Logical and Resource keywords will be added in a future release
  • Fixes an issue where GoFSH generated invalid contains rules when sliceName is specified in the differential on a choice slice
  • Fixes an issue where rules with references to contained resources were generated incorrectly

See the Release Notes for additional details.
