Topic: SUSHI 2.1.0

Nick Freiter (Oct 08 2021 at 16:54):

Introducing SUSHI 2.1.0. This release includes the following enhancements and bug fixes:

  • Automatically populate url element on Usage: #definition Instances
  • Warn when a concept is included in a ValueSet more than once
  • Support case-insensitive package names
  • Display source information when warning about ValueSet filter operand types
  • Support finding Usage: #definition Instances with InstanceOf: CodeSystem when resolving codes in FSH
  • Replace errors related to slice cardinality being greater than cardinality of the sliced element with a warning
  • Fix bug with validation of choice slices when validating cardinality of Instance elements
  • Update vulnerable dependencies

See the release notes for full details.
