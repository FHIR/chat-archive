Chris Moesel (Apr 07 2020 at 13:04):

If you want the full build, from FSH-to-IG to happen in auto-build, then you need to follow the new publishing approach documented here.

You still have the option of following other approaches, but this means that you'll need to check in the SUSHI output (which is IG Publisher input) into a repo configured for the IG Auto-build and ensure that you copy it into the root folder. The reason the automated workflow is the way it is is because the IG Pub / Auto-build really want the sources to be in the root folder.
