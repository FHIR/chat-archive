Topic: SUSHI 1.3.0

Nick Freiter (Mar 03 2021 at 19:52):

Announcing SUSHI 1.3.0 with the following new features, enhancements and bug fixes:

  • Add capability for SUSHI to expand certain ValueSets
  • Support all whitespace characters on quoted codes
  • Support targeting an Instance with the Canonical keyword
  • Add a warning that is logged when resources are specified with null array values
  • Fix referencing extensions by URL on Instances
  • Fix overwriting of extension URL values on Instances
  • Fix assignment of values to sliced children of an element
  • Fix assignment of extensions to primitive elements
  • Fix confusing error message observed on FSH Online
  • Fix bugs causing repeated extensions to appear on Instances

See the SUSHI 1.3.0 Release Notes for more details.
