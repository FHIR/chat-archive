Topic: SUSHI 0.4.1

Chris Moesel (Dec 21 2019 at 03:18):

I'm pleased to announce that today we released SUSHI 0.4.1 (and also 0.4.0, but you should really use 0.4.1!). The primary new features, changes, and fixes include:

  • Simplified support for adding extensions to profiles
  • Initial (limited) support for generating Implementation Guides
  • Default output folder changed from out to build
  • Bug fix for applying constraints to extension values
  • Reports total number of errors and warnings after processing completes

See the release notes for details:
https://github.com/FHIR/sushi/releases/tag/v0.4.1
