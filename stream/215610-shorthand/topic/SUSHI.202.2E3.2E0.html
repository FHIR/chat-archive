Topic: SUSHI 2.3.0

Julia Afeltra (Feb 25 2022 at 20:25):

Announcing SUSHI 2.3.0 with the following enhancements and bug fixes:

  • Improve warning message about files in unsupported folders within the input directory
  • Add support for specifying additional resource paths to predefined resources
  • Include provided extensions from sushi-config in the ImplementationGuide resource
  • Create an IG URL for dependencies listed in sushi-config that do not have an ImplementationGuide resource
  • Support referencing the canonical URL of the ImplementationGuide using the Canonical keyword
  • Improve SUSHI’s efficiency when processing profiles and eliminate unexpected error
  • Improve support for processing large base64 encoded values
  • Update the copy utility used during the build process

Install or Update

To install or update to this release, run the following command:
npm install -g fsh-sushi

More Information

For more details on this release, see the SUSHI 2.3.0 Release Notes.
