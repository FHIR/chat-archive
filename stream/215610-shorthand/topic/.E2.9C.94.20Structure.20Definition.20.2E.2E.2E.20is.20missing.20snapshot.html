Matthijs van der Wielen (Nov 02 2021 at 09:00):

@Mareike Przysucha I would recommend using Firely Teminal to install you packages. The latest release of Firely Teminal has a "bake" functionality in there that automatically generates snapshots for all the packages you want to install. You can install Firely Terminal easily on your command line https://simplifier.net/downloads/firely-terminal

The benefit of using Firely Terminal to generate the snapshots is that you always use the latest version of the .NET SDK snapshot generator.
