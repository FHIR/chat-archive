Hank Lenzi (Jan 19 2022 at 15:03):

Hello --

We want to apply the following restriction to the "performer" on the Immunization Resource ( https://www.hl7.org/fhir/immunization.html ):
There must be a performer and it is to be restrained to the PractitionerRole.

// performer 0.. Type: BackboneElement
//      +--------------> function 0..1 Type: CodeableConcept Immunization Function Codes (Extensible)
//      +--------------> actor 1..1 Type: Reference(Practitioner| PractitionerRole | Organization)

So, we do (we ONLY do this):

* performer.actor only Reference(PractitionerRole)

Is this correct?

TIA,
Hank
