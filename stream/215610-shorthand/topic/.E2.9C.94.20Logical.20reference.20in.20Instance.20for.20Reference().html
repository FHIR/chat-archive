Topic: ✔ Logical reference in Instance for Reference()

Richard Stanley (Jan 19 2022 at 22:54):

Hi. I'd like to use a logical reference in an instance. It's for Location.managingOrganization and I'd like to refer to an identifier of an Org rather than an Id. A minimum broken example is here: https://fshschool.org/FSHOnline/#/share/3GMdVmC and below. Is there a way to do that? Thanks!

Alias: $synthea = https://github.com/synthetichealth/synthea

Instance: Org-Simple
InstanceOf: Organization
Title: "Org-Simple"
Description: "Org-Simple"
* name = "Org-Simple"
* identifier[+].system = $synthea
* identifier[=].value = "1234"

Instance: Loc-Simple
InstanceOf: Location
Title: "Loc-Simple"
Description: "Loc-Simple"
// works
* managingOrganization = Reference(Org-Simple)
// doesn't work
// *  managingOrganization.identifier.system = $synthea
// *  managingOrganization.identifier.value = b8cc8bf5-7f24-599b-80d4-bd0cf274021f

Chris Moesel (Jan 19 2022 at 23:14):

Hi @Richard Stanley. I think it wasn't working because managingOrganization.identifier.value needs to be a string. Try:

*  managingOrganization.identifier.value = "b8cc8bf5-7f24-599b-80d4-bd0cf274021f"

Richard Stanley (Jan 19 2022 at 23:24):

Thanks! Indeed, that was it.
