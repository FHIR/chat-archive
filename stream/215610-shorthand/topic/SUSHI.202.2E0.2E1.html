Topic: SUSHI 2.0.1

Nick Freiter (Aug 25 2021 at 18:06):

Announcing SUSHI 2.0.1 which includes the following enhancements and bug fixes:

  • Support downloading packages via https rather than http
  • Support including version information in only rules on Canonical types
  • Fix bug causing soft indexing to fail on indented rules which are not path rules
  • Fix bugs in sushi --init causing IG Publisher to build with errors
  • Fix bug causing duplicate detection to only work within a file
  • Fix bug allowing duplicate concepts to be added to CodeSystem definitions via insert rules

See the Release Notes for more details.
