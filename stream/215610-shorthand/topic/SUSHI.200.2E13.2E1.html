Announcing SUSHI 0.13.1

SUSHI 0.13.1 is a minor patch release on the major 0.13.0 release. Users upgrading directly to 0.13.1 should still read the SUSHI 0.13.0 release notes. SUSHI 0.13.1 contains the following fixes:

  • Fixes a bug which causes SUSHI to hang when duplicate file names were present
  • Fixes a bug which caused an error when no ig-data folder was present

Full release notes: https://github.com/FHIR/sushi/releases/tag/v0.13.1
