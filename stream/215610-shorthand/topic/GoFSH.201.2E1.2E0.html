Topic: GoFSH 1.1.0

Nick Freiter (Apr 23 2021 at 18:47):

Announcing GoFSH 1.1.0 with the following enhancements and bug fixes:

  • Update returned content of fhirToFsh function when style is set to map so that the content can be treated as JSON content (#103)
  • Add support for generating FSH using the Reference keyword when appropriate (#104)
  • Add support for generating FSH using syntax for representing Quantities that do not use UCUM codes (#105)
  • Add support for generating FSH using soft indexing (#106)
  • Add support for inferring any missing configuration properties if possible (#108)
    See the release notes for more details.
