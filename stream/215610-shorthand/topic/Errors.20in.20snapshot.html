Chris Moesel (Jun 17 2021 at 18:22):

Hi @Kirstine Rosenbeck Gøeg -- by default, SUSHI only generates the differential portion of a StructureDefinition. The only way that SUSHI generates a snapshot is if you explicitly tell it to via the -s or --snapshot flag. Assuming you're not using that, then it is the IG Publisher that is generating that snapshot. This sounds like an IG Publisher issue to me, but if you're able to include a snippet of FSH that reproduces this, we can look into it further (as it could be a SUSHI issue).

Chris Moesel (Jun 17 2021 at 20:48):

It looks like this might be a more widespread bug in the publisher. See: https://chat.fhir.org/#narrow/stream/179252-IG-creation/topic/Canonical.20Errors.20on.20latest.20IG.20Publisher
