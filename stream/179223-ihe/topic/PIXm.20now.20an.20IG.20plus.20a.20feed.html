John Moehrke (Aug 20 2021 at 13:27):

Public-Comment period until end of August for the IG published PIXm (Patient Identity Cross-Reference for mobile)
https://profiles.ihe.net/ITI/PIXm/index.html
There is no intended change to what was previously defined. THERE is a new feature for a patient identity feed, the functional equivalent of the PIX feed. This feed is essentially just a RESTful Create/Update/Delete interface on the PIX registry.

Patient Identifier Cross-reference for Mobile (PIXm) - Rev. 3.0.0. This revision was created using the Implementation Guide publisher. This is an update of PIXm that was previously published in PDF form. This new publication is published in HTML form with a full set of conformance resources and examples. New functionality has been added to support a patient identity feed to inform the Patient Identity Manager of changes to patient identities.
https://profiles.ihe.net/ITI/PIXm/index.html
