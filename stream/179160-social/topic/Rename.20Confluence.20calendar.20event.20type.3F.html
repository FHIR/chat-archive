Select "Edit" at top right of the calendar > Event Types (on left) > Select the pencil icon next to the custom event type at the bottom of the list.

Joshua Procious (Jul 30 2020 at 11:15):

From here just edit and save then "ok".

Joshua Procious (Jul 30 2020 at 11:15):

image.png

Joshua Procious (Jul 30 2020 at 11:17):

Now there are a few standard event types... one of them being "Events". These cannot be renamed this way. So first we needed to create a new custom event type (which is found in that first edit screen at the very bottom) and name it. Then we go to the calendar itself and select one of the entries for the "Events" type.

Joshua Procious (Jul 30 2020 at 11:17):

image.png

Joshua Procious (Jul 30 2020 at 11:18):

Select "Edit" from the popup

Joshua Procious (Jul 30 2020 at 11:18):

image.png

Joshua Procious (Jul 30 2020 at 11:19):

Make sure "Edit this instance only" is unchecked then select the new even type at the top drop down and finally select "OK".

Joshua Procious (Jul 30 2020 at 11:20):
