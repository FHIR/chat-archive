Liam (May 10 2019 at 13:41):

Does fhir support 3 level queries. Like if we want to include practitioner which is linked to procedure request and here procedure request will be linked to procedure for specific patient .The query should look like this :
baseURL/STU3/Procedure?patient=patientID&_include=Procedure:basedOn&_include=ProcedureRequest:requester.
Here Procedure:basedOn will include procedure request and ProcedureRequest:requester will include Practitioner.
Does FHIR standard support that?

Lloyd McKenzie (May 10 2019 at 14:52):

Yes, though I believe you need to use iterate. E.g. baseURL/STU3/Procedure?patient=patientID&_include=Procedure:basedOn&_include:iterate=ProcedureRequest:requester
