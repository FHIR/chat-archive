Topic: How2do Confluence "Action Items" list

Dave deBronkart (Mar 17 2020 at 21:07):

This page explains how to create a Task in a Confluence page e.g. a WG's minutes. But then the page waves its hands and shows a populated list of such things (under a heading Action Items). How does on make this magic happen? Does the help page cruelly omit the words "For instance, you could use this feature to manually create a list of Action Items"?

Vassil Peytchev (Mar 17 2020 at 21:12):

When you edit, simply select the checkbox from the menu, then @mention the assignee
Multiple such tasks can be formatted as an Action Item list - there is nothing special to it, just a heading and list of tasks...
I just did it on the minutes page, to confirm that it works, feel free to delete it.

Dave deBronkart (Mar 17 2020 at 21:22):

Aha! So the checkbox can be added automatically, but contrary to what that help page looks like, there is NOT a feature that automatically grabs all the tasks from the page and lists them (like a Table of Contents).

I object. :-) But that's what I needed to know - thanks!
