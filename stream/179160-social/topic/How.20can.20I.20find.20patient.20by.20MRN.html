Topic: How can I find patient by MRN

Peter (Dec 05 2017 at 08:47):

Hi, there. I am a newbie at FHIR. I want to search patients by their MRN.
As I understand identifier is MRN if it has type system "http://hl7.org/fhir/v2/0203" and code "MR".
But how can I find a patient by them?

Lloyd McKenzie (Dec 05 2017 at 15:54):

This would probably be better raised on the implementer stream. At the moment, there's no standard search criteria that lets you search by identifier type + value, only system + value. You could define your own composite search criteria. You could also submit a change request for us to define a standard mechanism.
