Ward Weistra (Feb 12 2021 at 09:17):

There's two useful Zulip formatting features I just learned about:

Timezone aware time
Start typing <time and a datetime picker shows up. Select a date and time and it will be displayed for everyone in their local timezone.
For example: The next FHIR-I call starts at 2021-02-15T21:00:00+01:00.

Spoiler block
While officially meant for spoilers it can be abused to have a collapse/expand block in your text for any reason:

This is how you create one

Just create a regular code block, but set spoiler as the 'language'. The text on the same line will be the preview text.
```spoiler This is how you create one <- Begin of block
Some content
End again with three backticks (`).
