not sure this is social, but does anyone have any info on mapping FHIR onto the FAIR Guiding Principles i.e. https://www.nature.com/articles/sdata201618 (and https://en.wikipedia.org/wiki/FAIR_data). Grahame Grieve (Jun 04 2019 at 00:38):

Rob Mulders (Jun 12 2019 at 18:00):

Hi @Rik Smithies I haven't done any mapping, but HL7 Netherlands has good contact with the GoFair consortium. Actually, on the HL7 Netherlands WGM last year, we had a small hackathon about using a FHIR server as data station in a Personal Health Train research algorithm that was sent to multiple FAIR data stations. The source code for using a FHIR server as data station is at https://github.com/jvsoest/PHT_on_FHIR_demo
