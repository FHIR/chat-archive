Hello. I am looking for a download file for FHIR code systems used in US Core 3/R4. I presume since R4 is a normative version, there is a download somewhere? I've searched on the FHIR downloads page and looked in NLM's Value Set Authority Center but did not find them. I'm not interested in any of the other code systems (SNOMED, RxNorm, HL7 V2/V3, etc.) but would like a file for FHIR code systems. Thanks in advance.

Lloyd McKenzie (Apr 28 2021 at 22:31):

There is no single download of code systems referenced by IGs, only code systems defined within the IG itself. For code systems defined on terminology.hl7.org, you can either grab the XML or JSON definitions individually on a system by system basis, or you can download the definitions for all terminology.hl7.org systems.
