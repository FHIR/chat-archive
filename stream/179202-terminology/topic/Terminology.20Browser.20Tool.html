As a means of making the content on a terminology server a little more accessible. We have been developing a tool to allow the CoseSystems, ValueSets, and SNOMED refsets to be viewed.
The "TermCat" tool can be used to visualise the data in a FHIR compatible server that supports the "Terminology Services" interface as defines in FHIR (STU3 or R4).

If you have the time them take a look at https://termcat.netlify.app/
This is still in development so functionality will evolve and it may be unstable from time to time.
For info, this tool is currently configured to use the public OntoServer hosted by CSIRO
