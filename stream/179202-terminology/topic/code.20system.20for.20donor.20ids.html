Bob Milius (Apr 08 2020 at 21:12):

We need a code/naming system for the "Global Registration Identifiers for Donors" (GRID).
The page describing it is found here
https://www.iccbba.org/tech-library/iccbba-documents/standards-documents/global-registration-identifier-for-donors-ion-database-and-grid-rules

How our organization uses it is found here:
https://www.cibmtr.org/DataManagement/TrainingReference/Documents/GRID%20Frequently%20Asked%20Questions%20Network%20CIBMTR.pdf

The GRID consists of
4-digit Issuing Organization Number
13-character Registration Donor Identifier (RDI)
2-digit checksum

We would like to use it a patient identifier in our donor registry, but wonder what code/naming system we would give it?
