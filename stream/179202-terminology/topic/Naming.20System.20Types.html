Topic: Naming System Types

Peter Jordan (Jan 14 2019 at 20:28):

Following from the discussions about the NamingSystem resource in Q2 this morning, the Vocab WG is keen to identify uses of the 'codesystem' code for NamingSystem.kind. The purpose of this is to determine if this value is actually necessary or merely replicates information that could be placed in the identifier attribute of the relevant Code System resource. In NZ, we are using this kind of NamingSystem resource as an easy means of translating Code System OIDs used in CDA documents to URIs for FHIR - and it's also a lightweight means of providing basic details of NZ-specific code systems with URIs that are resolvable to Web Page descriptions.
