Huib (Aug 18 2020 at 15:06):

Hi,

If the external terminology server you are using is https://fhir.loinc.org this result is expected, because the actual expand is done by that server instead of Vonk.
I quickly tested https://fhir.loinc.org/ValueSet/$expand?url=http://loinc.org/vs&filter=SARS-CoV-2 and got the same result.

The LOINC FHIR server is still in Beta, so they might be able to fix this if you contact them.
