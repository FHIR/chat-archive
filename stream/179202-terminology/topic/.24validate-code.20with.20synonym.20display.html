Topic: $validate-code with synonym display

Yunwei Wang (Sep 18 2018 at 18:34):

While implementing $validate-code for CodeSystem resource, we met a question about SNOMED CT synonym. "Using SNOMED CT in FHIR" states that "The correct display for a SNOMED CT concept is one of the terms associated with that concept." Using SCT 8762007 as example, this concept has Preferred FSN "Chronic hypertension in obstetric context (disorder)" and Preferred Synonym "Chronic hypertension in obstetric context".
Questions 1: Should both opeations

$validate-code?system=http://snomed.info/sct&code=8762007&display=Chronic%20hypertension%20in%20obstetric%20context%20(disorder)

and

$validate-code?system=http://snomed.info/sct&code=8762007&display=Chronic%20hypertension%20in%20obstetric%20context

return true?
Question 2: How about ValueSet.$validate-code? If a value set has this code and FSN display, what is the result for $validate-code with synonym display?
