Carol Macumber (Sep 15 2021 at 18:11):

Per HTA JIRA Ticket here https://jira.hl7.org/projects/HTA/issues/HTA-47

HTA has registered the CDC Race and Ethnicity Code System (details here https://confluence.hl7.org/pages/viewpage.action?pageId=77369019) and, as implemented already in US Core and at the NLM's VSAC, recommends the use of the URN formatted URI = urn:oid:2.16.840.1.113883.6.238 as it's code system identifier.

Next steps will be to update the code system entry at terminology.hl7.org to ensure alignment with this information.

Please contact the HTA or comment on the above mentioned Jira ticket if necessary.
