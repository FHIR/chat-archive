Paul Lynch (Jun 21 2019 at 21:45):

Is there a public FHIR server that supports $expand on some ValueSets? I am trying to find something to experiment with, and don't really care which ValueSets are included. I've checked http://wiki.hl7.org/index.php?title=Publicly_Available_FHIR_Servers_for_testing, but could not get $expand to work on any listed as supporting terminology.

Rob Hausam (Jun 21 2019 at 22:13):

The public HAPI server (which is listed) does it, and any of the terminology servers should be able to do it (it's arguably the most fundamental terminology service operation). They aren't actually listed, but the tx.fhir.org terminology servers used by the FHIR build (i.e. tx.fhir.org/r4) definitely do $expand (test.fhir.org is listed, but not tx.fhir.org for some reason). If it still doesn't work for you let us know.
