Topic: Invalid UCUM units

Grahame Grieve (Mar 25 2018 at 09:10):

Some time ago, I set the build to produce an error message if it thought a UCUM code wasn't valid. I'm just going through them now. I think that the following UCUM codes are not valid:

  • '{Ehrlich'U}/100g (should be '{Ehrlich'U}/(100.g)')
  • [M'U] - just fictional
  • '{G}U' - should be '{G}.U'
  • mmol/kg[H20] - I'm unclear on this. UCUM allows cm[H2O], but I cannot find g[H2O] defined anywhere

does anyone agree with this?
