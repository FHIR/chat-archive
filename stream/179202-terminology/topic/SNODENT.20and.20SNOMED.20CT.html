Robert McClure (Aug 13 2020 at 21:01):

@Michael Lawley SNODENT and SNOMED CT are inter-related and all SNODENT concepts also appear in SNOMED CT but each code system has a code system-specific code. All SNODENT codes end in a (capitalized) letter so I suspect you are seeing the SCT concept identifier if it looks like one. Take a look at Bloodstained sputum (finding) which you will note in the url, actually uses the SCT id for the url. None-the-less, if you are identifying the concept as a SNODENT concept (as is necessary for interoperability on dental information) you need to use the SNODENT ID, in the case of the example, that would be 148994D. @Jean Narcisi is the one in charge and she will fix any misstatements I've made.
