Michael Lawley (Oct 17 2019 at 18:30):

What I pointed to is a concept status not the CodeSystem.status. The way to use is is to add a property called "status" with value = "deprecated" to each of your deprecated codes. You'll also want to declare that property in the CodeSystem using the appropriate URI http://hl7.org/fhir/concept-properties#status
Here's an example

{
   "resourceType" : "CodeSystem",
   "name" : "DeprecationExample",
   "text" : {
      "status" : "generated",
      "div" : "<div xmlns=\"http://www.w3.org/1999/xhtml\"><h2>DeprecationExample</h2><tt>urn:code-deprecation</tt><p></p></div>"
   },
   "meta" : {
      "versionId" : "1",
      "lastUpdated" : "2019-10-18T04:29:40.245+10:00"
   },
   "url" : "urn:code-deprecation",
   "content" : "complete",
   "compositional" : false,
   "count" : 3,
   "id" : "9d5e05f6-af76-45e8-8dd7-35032bb50950",
   "status" : "draft",
   "property" : [
      {
         "code" : "status",
         "type" : "code",
         "uri" : "http://hl7.org/fhir/concept-properties#status"
      }
   ],
   "concept" : [
      {
         "display" : "OLD",
         "code" : "old",
         "property" : [
            {
               "code" : "status",
               "valueCode" : "deprecated"
            }
         ],
         "definition" : "my old code"
      },
      {
         "display" : "NEW",
         "definition" : "my new code",
         "code" : "new"
      }
   ]
}
