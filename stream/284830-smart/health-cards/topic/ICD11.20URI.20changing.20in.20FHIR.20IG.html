Topic: ICD11 URI changing in FHIR IG

Max Masnick (Aug 06 2021 at 17:04):

Per https://confluence.hl7.org/display/TA/International+Classification+of+Diseases+11th+Revision, the URI for ICD11 has changed from "http://id.who.int/icd11/mms" to "http://id.who.int/icd/release/11/mms". I'm not aware of any implementers using ICD-11 so I'm planning to make this change prior to ballot, which will shortly be reflected in the main branch of the FHIR IG.

If this is a problem for anyone please let us know.
