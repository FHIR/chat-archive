A new version of the FHIR IG is released! (v0.5.0-rc)

  • Release notes: https://github.com/dvci/vaccine-credential-ig/releases/tag/v0.5.0-rc
  • CI build: https://vci.org/ig/vaccination-and-testing/

This is the last RC before our official stable release. We will be accepting bug reports until Friday, May 21 for anything that needs to be fixed before the stable release goes out. We will hold feature requests for a subsequent release unless they are absoltuely critical for our core use case.

There are a number of breaking changes in this release, but most are minor. Please let us know if there are any unexpected or problematic breaking changes (again, by Friday, May 21) -- this is still a RC, so there is still time to put in fixes before the final v0.5.0 release.
