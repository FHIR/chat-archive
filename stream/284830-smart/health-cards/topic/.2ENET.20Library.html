Here is an open-source MIT Licence .NET 5 Smart Health Card (JWS) token and QR Code encoder, decoder and verifier library with accompanying Nuget packages which I have been working on.

It does not validate your FHIR bundle but rather accepts JSON as the payload and appropriately encodes and decodes it to and from the token. I would suggest using the FIRELY .NET SDK for the task of bundle creation.

Here it is with examples of use: SMART Health Card JWS token and QR code generation library
