Mareike Przysucha (Jul 30 2020 at 15:13):

This would normally be an identifier.

As an identifier needs a system, it is advisable that you have a look at what there already is in your country. I write this because to increase interoperability it is better if everyone uses the same "system"-URI if refering to the same real system.
If I get your eMail-address correctly, you're from France, is that right? In this case (a) there is a profile on simplifier for French patients (https://simplifier.net/frenchprofiledfhirar/frpatient) and (b) there is a stream for France where you can ask about France-specific matters, e.g. national identifiers. If I'm wrong about France, I apologize.
