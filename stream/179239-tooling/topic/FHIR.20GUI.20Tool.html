David Hay (Oct 20 2020 at 16:40):

Sure - it's a web based tool. If you want to use another server then it can be added in the box at the top right. The Patient Viewer is probably the module you want. If you need an access token then it can be added in the patient selection screen. Currently HTTP only, but will add https shortly. There's a separate chat stream for questions - https://chat.fhir.org/#narrow/stream/179176-clinFHIR

David Hay (Oct 20 2020 at 16:44):

Oh, and there's also a module that allows you to create instance graphs for design / education. There have actually been a number of iterations of this, the latest is incorporating sushi to help with the heavy lifting as it handles profiles much better than I do - almost ready for release...
