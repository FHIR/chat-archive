Topic: Transforming openEHR archeytpes to FHIR Observation resource

Diego Bosca (Oct 30 2019 at 11:50):

Latest LinkEHR version has the functionality to transform openEHR archetypes to FHIR R4 Observation resources. Process generates the Observation and ValueSet profiles from a given archetype, and the Extensions if any archetype element has been selected as that. Here is a little blog post with the process + current limitations https://standardsmania.blogspot.com/2019/10/using-linkehr-to-generate-fhir.html
LinkEHR can be freely downloaded at linkehr.com
