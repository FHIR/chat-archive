Topic: Torinox is now Firely Terminal (and with more features)

Ward Weistra (Dec 09 2020 at 17:26):

Firely's cross-platform FHIR command line tool Torinox has received a new name: Firely Terminal!
Download for free via these instructions: https://simplifier.net/downloads/firely-terminal

I've recorded an overview of the (new) functionality here: https://www.youtube.com/watch?v=ukD9VR0lOVM

New functionality includes:

  • ValueSet Expansion
  • Querying over all your FHIR resources with the Firely Query Language

Other features:

  • A full FHIR client
  • FHIR validation
  • Synchronization with Simplifier.net
  • Installing FHIR packages Package
  • Snapshot generation
  • XML/JSON conversion
  • Bundle creation and splitting

@Martijn Harthoorn has been tuning this one for years, so please let him know your thoughts :smile:

Ward Weistra (Dec 09 2020 at 17:26):
