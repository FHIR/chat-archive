Topic: Making Forge run on Catalina

Morten Ernebjerg (Nov 15 2019 at 10:16):

Hi @Michel Rutten and @Ward Weistra I saw that in the Forge install guide section on macOS explicitly states that Forge will not run with Wine on macOS Catalina. Are there plans to make this possible (or maybe someone knows a workaround...)? It would be great for us mac people, especially since not updating the OS may not be an option due to corporate/institutional policies.

Ward Weistra (Nov 15 2019 at 10:21):

@Morten Ernebjerg Yes, we'll surely try to make this work later this year. We're a victim of Catalina not supporting 32-bit applications anymore and Forge being compiled as such.

In the meantime I assume that running Windows in a VM on Mac and running Forge on that should still work for Catalina users, but this does require a Windows license.

Morten Ernebjerg (Nov 15 2019 at 10:23):
