@Patrick Werner the file Messages_de.properties is now about 50 messages behind

Everyone else: the validator now includes a framework for internationalising error messages. So far, only german (de) translations have been provided. If you want to provide translations to a different language, you need to create and maintain a sibling to this file: https://github.com/hapifhir/org.hl7.fhir.core/blob/master/org.hl7.fhir.utilities/src/main/resources/Messages_de.properties, which is a language variant to the master file https://github.com/hapifhir/org.hl7.fhir.core/blob/master/org.hl7.fhir.utilities/src/main/resources/Messages.properties. I only ever add things to the end of the master file - even if I revise an existing message, I move it to the end of the file, to make it easy for maintainers.

Translations for other languages are most welcome. @Peter Jordan can you please pass this on to the intl co-chairs, so they can see if someone wants to take this task on on behalf of the affiliate? (required knowledge: some knowledge of FHIR, and how to use github)
