Topic: Forge 24.1 for STU3 & R4

Abel Enthoven (Mar 17 2020 at 12:36):

Forge 24.1 has just been released and is now available via https://simplifier.net/forge or auto-update!
Forge 24.1 implements support for technical corrections 3.0.2 and 4.0.1. New profiles are automatically assigned the updated versions and opened profiles are updated automatically.
In addition to that, the new Forge versions fix a few bugs regarding slicing and now retain any manually added StructureDefinition extensions. For a full list of fixes, please see Forge's own readme file (shown after install or update). For known issues, please check: https://simplifier.net/docs/forge/knownissues. For questions and feedback, please reach out at forge@fire.ly.
