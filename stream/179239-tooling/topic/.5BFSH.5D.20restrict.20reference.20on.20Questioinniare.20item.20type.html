Topic: [FSH] restrict reference on Questioinniare item type

Christian Nau (Apr 08 2021 at 08:12):

Hello,
I'm defining a questionnaire using FSH. The questionnaire contains an item with item.type = #reference. Is it possible to restrict references to be of type Practitioner?
From defining profiles I know of the possibility to use value[x] only Reference(Practitioner).

Chris Moesel (Apr 08 2021 at 12:10):

Hi @Christian Nau. The base resource only allows you to say the item type is a reference, but you can use an extension to provide the required resource type. I actually didn't know this either, but I looked through the core FHIR extensions, searching for extensions with context Questionnaire.item, and found the questionnaire-referenceResource extension.

To use it in FSH, you would do something like this:

Alias: QREF = http://hl7.org/fhir/StructureDefinition/questionnaire-referenceResource

Instance: MyQuestionnaire
InstanceOf: Questionnaire
* status = #draft
* item[0].linkId = "1"
* item[0].type = #reference
* item[0].extension[QREF].valueCode = #Practitioner
