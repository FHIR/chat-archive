Hello Team, Is there a public FHIR server that I can use to test the Program Edition of the Inferno Tool? Ideally this public FHIR server is able to pass most of the certification test cases. Robert Scanlon (Feb 11 2021 at 20:32):

Yes, we provide https://inferno.healthit.gov/reference-server/ which passes all the authorization & single patient api tests. We also have a bulk data server for the multi-patient api tests.

Robert Scanlon (Feb 11 2021 at 20:33):

The easiest way to run it is to choose the 'Inferno Reference Server' preset on https://inferno.healthit.gov/inferno (look below the input for the '...or choose a public sandbox')

Lakshmi Bhamidipati (Feb 11 2021 at 20:34):

Thank you @Robert Scanlon . I'll look into it.

Robert Scanlon (Feb 11 2021 at 20:35):

One thing to note is you will need to get a bearer token from one of the three 'App' tests (authorization) before you run the single patient api tests. So I just run the 'Standalone Patient App' test, click through the screens until it all passes, then go to the Single Patient API tests which will now have the Bearer token filled in.
