James Threatte (Oct 16 2020 at 21:38):

I was reviewing the Inferno 1.2 and 1.3 matrices, and noticed there are test IDs ending in -02 that are listed on the Test Procedure sheet but are not shown on the Inferno Test sheet. The list of tests that are missing is

USCAI-02
USCC-02
USCCP-02
USCG-02
USCI-02
USCID-02

Does this have something to do with them being -02? What is -02?

Yunwei Wang (Oct 20 2020 at 14:57):

@JamesT Those skipped tests are optional tests which are not required by US Core IG. For example, USCAI-02 tests search combination of patient and clinical-status. According to US Core IG (http://hl7.org/fhir/us/core/STU3.1/StructureDefinition-us-core-allergyintolerance.html#optional-search-parameters), "(server) SHOULD support searching using the combination of the patient and clinical-status search parameters".
These optional tests are not included in Inferno-Program edition. If you would like to run optional tests, you could use Inferno-Community edition.
