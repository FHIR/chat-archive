Yunwei Wang (Jan 19 2022 at 14:56):

Hello Stephen, Inferno provide a GenericGenerator to generate tests from StructureDefinitions. What you need to do is,

  • unzip IG to /resources/[ig folder name] folder
  • run command bundle exec rake:generate['generic','ig folder name']
    For example, if I want to generate tests for IPS without any special treatment, then I copies IPS to /resources/ips folder and run command bundle exec rake:generate['generic','ips']
    If you need special treatment, then you need to create your own generator inheriting from Base generator. There are three generators under /generator folder which demo how to create your own generators.
    Please let me know if you have more questions.
