https://davinci-pcde-client.logicahealth.org/memberMatch This client can be used to test any $member-match implementation. The URL that is used can be specified under the FHIR Endpoint. It will post to that URL with the added 'Patient/$member-match' appended. The messages are somewhat limited, but after filling out the fields you can generate a sample message of what would be posted with the Sample Message button. The Select Member button will send that message to the specified endpoint
