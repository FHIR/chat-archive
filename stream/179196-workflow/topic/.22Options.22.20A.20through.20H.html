Patrick Werner (Oct 12 2016 at 12:03):

the main difference seems to be the synchronous vs asynchronous interaction. Option A is synchronous and therfore has to respond during the http timeout period

Only works for automated execution where the decision to perform the request and the execution of the request can be done synchronously within the HTTP timeout period (generally on the order of 10s of seconds).

Option B has to Interactions:
1. placer places request, gets 201 -> i got the request
2. filler posts response to placer, gets 201 -> i got the response
