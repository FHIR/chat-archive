Topic: URL in an IG ValueSet

Brian Reinhold (Nov 23 2018 at 17:18):

How should I define the URL for a value set defined in an implementation Guide?
I have
<url value="http://hl7.org/fhir/uv/phd/ValueSet/String11073MDC"/>
but "http://hl7.org/fhir/uv/phd/" is from a previous version of this implementation guide and in that one there was no value sets. Now the guide exists at
build.fhir.org/ig/HL7/PHD
and the value set at build.fhir.org/ig/HL7/PHD/src/resources
but somehow that does not seem where it will be at the end of OR for ballot

Lloyd McKenzie (Nov 23 2018 at 23:09):

"http://hl7.org/fhir/uv/phd/" is the canonical root for your IG. It is version independent. That's what the canonical URL of all of your artifacts should start with - regardless of what version of the iG.
