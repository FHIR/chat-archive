Lloyd McKenzie (Oct 28 2021 at 14:55):

That involves creating your own set of templates - which is possible, though not straightforward and not always well documented (fixing the documentation issues is on our 'list of things to do'). You can find some documentation about IG templates here: https://build.fhir.org/ig/FHIR/ig-guidance/

It also has references to the various ig-template Git projects, which you can inspect to see how well they work.

Finally, there is an intention to introduce a more sophisticated mechanism to control how artifacts are organized - likely using Composition. However, if/how/when that will happen is not yet landed.
