Topic: Regarding the valuesets in IG

Mounika (Mar 29 2018 at 12:18):

Hi all, I want to add my value sets to the IG.How it is possible.I am using the vhdir IG.Thank You

Lloyd McKenzie (Mar 29 2018 at 14:45):

You can't add your own artifacts to someone else's IG. You can define your own IG that is based on someone else's IG and add your own artifacts to it and/or constrain profiles from the parent IG (which might involve making references to your own value sets if they are a legal constraint of the binding from the parent profile)

Mounika (Apr 02 2018 at 12:23):

@Lloyd McKenzie We are using vhdir as reference to create our own IG. Can you tell me how to add our valuesets to my own IG?

Lloyd McKenzie (Apr 02 2018 at 14:28):

Define the value sets, include the value sets as resources referenced in your IG. If you're having trouble with that process, can you be specific about whether you've already gone through the IGPublisher instructions on the wiki, whether you've looked at some of the other implementation guides hosted on github and, if so, specifically what problems you're encountering?
