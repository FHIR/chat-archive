Grahame Grieve (Jun 24 2020 at 03:20):

As of the next version of the IG publisher, it will produce 2 new files per resource in the _includes directory: {{{type}}}-{{{id}}}.json and {{{type}}}-{{{id}}}.escaped.json. These files are the json source of the resources so that you can include them in a page source to do javascripty things with them. The .json is native json. The .escaped.json file is html escaped, and suitable to use a string variable
