As from the next release of the IG publisher, the publisher will create 2 additional fragments:

  • [type]-[id]-ip-statements.xhtml - a list of appropriate statements around the copyright of content included in the example/profile etc. (not needed for ValueSets, which already manage this)
  • ip-statements.xhtml - a global statement of all use in any resource, suitable for inclusion in the about page etc

The actual copyright statement generation will be hooked up to the HTA process automatically; you won't need to do anything about this other than put these in your output.
