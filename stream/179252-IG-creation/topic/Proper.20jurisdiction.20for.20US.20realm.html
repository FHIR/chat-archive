Grahame Grieve (Apr 26 2020 at 03:58):

urn:iso:std:iso:3166:-2#US is correct.

Keith Boone (Apr 26 2020 at 04:08):

And furthermore:
http://www.hl7.org/fhir/iso3166.html says:
ISO 3166 is a 3 part standard, each of which defines a code system:
urn:iso:std:iso:3166: Part 1, Country Codes (2-letter and 3-letter, and numerical codes)
urn:iso:std:iso:3166:-2: Part 2, Subdivision (State/Province) Codes
urn:iso:std:iso:3166:-3: Part 3, Historical Country Names

So, I think you are incorrect. And the specification indicates that the code should be: urn:iso:std:iso:3166#US
