@sai charan Check that the first element in your differential refers to the base resource and that you have a node in your differential for each ancestor of any node listed in the differential. For example, if all you want to constrain is Patient.contact.name, you must still (for the current version of the IGPublisher) have a node in your differential for "Patient" and "Patient.contact".
