Without messing with the IG itself it is possible to alter the configuration of the template in use in config.json. Here an example for CodeSystem if you want to prevent TTL from being generated:

  "defaults": {
    "CodeSystem": {
      "ttl": false,
      "ttl-html": false
    }
  }
}
