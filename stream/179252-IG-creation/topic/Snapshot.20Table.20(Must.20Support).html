Is the assumption correct that only the elements with the flag mustSupport = true are shown in the Snapshot Table (Must Support) and required elements (min = 1) with mustSupport = false are not shown? Has this always been the case or has something changed?

Lloyd McKenzie (Oct 27 2021 at 13:14):

That is correct and has always been the case. There's a proposal to replace the mustSupport page with something different - that includes mandatory elements, modifier elements, elements with invariants not defined in the core resource and other elements of key interest to implementers, however that hasn't happened yet.
