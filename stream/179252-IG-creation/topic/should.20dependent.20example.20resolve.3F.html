Grahame Grieve (Oct 15 2019 at 18:31):

so... I don't think this should resolve. You need to refer to it by it's full URL - a relative URL isn't going to resolve.

e.g. a reference to Practitioner/practitioner-1 in http://www.fhir.org/guides/test3 is not a reference to Practitioner/practitioner-1 in http://hl7.org/fhir/us/core
