Grahame Grieve (Apr 08 2019 at 19:50):

we have no concrete examples but we made the change as part of a set of changes where the data types that were not general purpose datatypes used in more than a few places were change to specialise backbone element - Dosage, and Timing, in particular had known use cases. But I can imagine use cases where you want to say something about a data element that is not ignorable

Grahame Grieve (Apr 08 2019 at 19:50):

I don't think you need to support extensions or modifier extensions on Forge except by specific request
