Michel Rutten (Nov 23 2016 at 11:46):

Hi @Simone Heckmann, great question! Currently, Forge automatically manages the .extension elements internally. The .extension child elements are not exposed in the UI so you cannot define explicit constraints on them (e.g. cardinality, slicing). This is a known limitation that we want to improve in a future release.
