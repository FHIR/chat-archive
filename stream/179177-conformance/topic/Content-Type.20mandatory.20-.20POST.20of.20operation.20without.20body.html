Is a content-type header mandatory when performing an operation on the base, without a body, using a POST?

The spec does not really describe this situation in the table: https://www.hl7.org/fhir/http.html#summary
According to the the HTTP spec it is not mandatory: https://tools.ietf.org/html/rfc7231#section-3.1.1.5
