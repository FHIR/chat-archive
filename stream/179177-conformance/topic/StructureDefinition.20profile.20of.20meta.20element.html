Michel Rutten (Dec 21 2017 at 14:21):

Hi @John Moehrke, Forge hides common resource elements such as .meta by default, as they might confuse new users. The Options menu provides a setting under 3rd heading FHIR Options" / "Show common resource elements". Enabling this option allows you to specify profile constraints on e.g. the Resource.meta element.
