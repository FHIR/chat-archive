Noticed a problem with the latest version of Forge when saving or loading Procedure resources with the performed[x] element.

To recreate:
1. Open Forge, create a new constraint on the Procedure resource.
2. Don't modify anything with the resource.
3. Save the resource.
4. Close the resource.
5. Open the saved resource.
6. You will noticed the performed[x] element doesn't appear any longer.

Also have noticed a crash when trying to open Procedure resource containing a differential with the performed[x] element in it.
