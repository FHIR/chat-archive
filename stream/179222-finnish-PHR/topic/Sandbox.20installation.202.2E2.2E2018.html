Hi all,

the Sandbox environments will be updated at 2.2.2018 to new versions of applications and profiles. During the updating there might be momentary short breaks in the service. Change information can be found below.

1. Changes in Blood Glucose profile:
Blood Glucose measurement code is updated:
- The value is found from path Observation.code.coding.code
- Old code: 77135-2
- New code: 14743-9

Blood Glucose measurement unit is updated:
- The value is found from path Observation.valueQuantity.unit
- Old unit: mmol/L
- New unit: mmmol/l

2. The application doesn’t anymore allow to save new versions of profiles with same URLs. This restriction has been added to make it possible to validate the resources unambiguously. Instead of saving a new version the old version must be updated.
