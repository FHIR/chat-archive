FYI

This project https://github.com/NHSDigital/validation-service-fhir-r4 provides FHIR Validation as a service. It is designed to be used in the new EPS FHIR api (which uses UKCore, NHSDigital and EPS Implementation Guides)

It is based on HAPI FHIR and so uses similar code to the HL7 FHIR command line FHIR Validator.

It can be reused for other projects. I've started using it for testing UKCore, NHSDigital and EPS Implementation Guides which are github workflows.
(and a version exists which includes NHS Ontology Server for coding validation - see example workflow here https://github.com/NHSDigital/NHSDigital-FHIR-ImplementationGuide/blob/master/.github/workflows/terminology.yml)
