Richard Kavanagh (Mar 08 2017 at 09:19):

For UK people.. NHS Digital are now moving into development stage of the CareConnect API. This is a RESTful API using the CareConnect Profiles intended for use in a number of care settings (including Acute, mental Health & Community). Discussions relating to this will be happening on the INTEROPen Ryver site. So if this is a development you are interested in then take a look https://interopen.ryver.com/index.html#forums/1093815
