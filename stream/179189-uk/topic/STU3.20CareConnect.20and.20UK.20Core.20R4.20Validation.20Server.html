fhir.airelogic.com has been updated to latest HAPI version 4.2
Validation has been swapped over to use the HL7 packaging system and also supports R4 UK Core Validation.
For STU3 it is using package: careconnect.base.stu3 version: 3.0.0
For R4 its using an unofficial version: uk.testcore.r4 version 4.0.0
These packages should also work with the HL7 validator

To validate using R4 you will need to use the rest endpoint POST https://fhir.airelogic.com/ccri-fhir/R4/$validate
