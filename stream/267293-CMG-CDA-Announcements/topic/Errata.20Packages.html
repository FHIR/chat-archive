Lisa Nelson (Dec 10 2020 at 14:23):

On September 16, 2020 a security issue surfaced in the generic stylesheet for display. At the virtual HL7 WGM September 2020 a meeting was held to plan how the issue would be addressed. The detailed action items from that meeting can be found on confluence at https://confluence.hl7.org/display/SD/CDA+R2+Stylesheet
The stylesheet vulnerability was addressed and release 4.0.2-beta8 – 2020-09-21 contained the fix. All published CDA implementation guide packages were reviewed to determine if the vulnerable stylesheet had been included in the download package.
Errata releases have been published to remove the vulnerable stylesheet and reference the new GitHub repository where the corrected stylesheet is now managed. This change impacted seven CDA R2.0 implementation guides. Implementers are encouraged to update their download package for these Implementation Guides and discontinue use of stylesheets versions prior to 4.0.2-beta8 - 2020-09-21.
