CDA R2.1 Errata Package Published on January 17, 2021

On January 17, 2021, the CDA Management Group announced the release of an errata update for the CDA R2.1 base standard. 

A technical correction has been made to fix a linkage issue between the CDA schema files and a new download package including the errata correction is now published.

For implementers who are testing CDA R2.1, the updated CDA R2.1 Schema with this errata correction can be found using this permalink: https://hl7.org/permalink/?CDAR2.1schema
