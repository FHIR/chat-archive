Stream: CMG CDA Announcements

Topic: CDA R2.0 Schema and CDA Stylesheet moved to HL7 GitHub

Lisa Nelson (Dec 10 2020 at 14:27):

On December 7, 2020 CMG and SDWG jointly announced the location of the CDA R2.0 Schema with SDWG approved extensions for the sdtc: namespace has moved to a new HL7 GitHub repository. The generic CDA Stylesheet for CDA R2.0 and CDA R2.1 has also moved to a new HL7 GitHub repository.
Permalinks have been established for the CDA R2.0 Schema and CDA Stylesheet which can be used to reference these new repositories.
https://hl7.org/permalink/?CDAStyleSheet
https://hl7.org/permalink/?CDAR2.0schema
For implementers who are testing CDA R2.1, the initial draft of the CDA R2.1 Schema based on the 12/6/2019 publication can be found using this permalink: https://hl7.org/permalink/?CDAR2.1schema
