CDA Implementation Guides Now Using JIRA for Balloting as well as Specification Feedback

CDA Management Group is pleased to report the January 2022 Ballot will include four CDA Implementation Guides for review:

  • HL7 CDA® R2 IG: CCDA Templates for Clinical Notes STU Companion Guide, Release 3 STU1- US Realm (JIRA v3.1.0-ballot)
  • HL7 CDA® R2 Implementation Guide: Consolidated CDA Templates for Clinical Notes; Advance Directives Templates, Release 1 STU2 - US Realm (JIRA v1.2.0-ballot)
  • HL7 CDA® R2 Implementation Guide: Emergency Medical Services; Patient Care Report, Release 3 Normative1 – US Realm (JIRA v3.1.0-ballot)
  • HL7 CDA® R2 Implementation Guide: ePOLST: Portable Medical Orders About Resuscitation and Initial Treatment, Release 1 STU1 - US Realm (JIRA v1.1.0-ballot)
    Each of these specifications will be available for making ballot comments via JIRA under the version indicated for the ballot, for ballot pool participants.

This is the first time the CDA Community will be using JIRA to management ballot comments.

New to JIRA for balloting?

If you need training on the new JIRA Ballot Process, information is available in Confluence: https://confluence.hl7.org/display/HL7/How+to+Guide+for+Balloting+using+Jira

The educational material includes a high-level overview of the overall HL7 ballot voting process in JIRA and how to set up your ballot desktop.

It also includes step-by-step instructions on how to sign-up for a ballot, submit an overall vote on a spec or change your vote on a specification. And it provides step-by-step instructions on how to submit ballot issues using JIRA.

For ballot reconciliation managers, the documentation also includes step-by-step instructions on how to create a reconciliation sheet.

Additionally, there are three CDA IG’s proposed to be withdrawn:

  • HL7 CDA® R2 Implementation Guide: S&I Framework Transitions of Care Companion Guide to Consolidated-CDA for Meaningful Use Stage 2, Release 1- US Realm
  • HL7 Implementation Guide for CDA® R2 - Supplement to Consolidated CDA for Attachments, Release 1- US Realm
  • HL7/ASTM Implementation Guide for CDA Release 2 – Continuity of Care Document- US Realm
    The January 2020 ballot pool closes on December 9th. Don’t forget to sign-up and participate in this important activity for creating and managing HL7 implementation guides throughout their lifecycle.
